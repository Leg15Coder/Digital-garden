# <h1 id="p1p1"><center>Выборка, вариационный ряд, оценка</center></h1>

Математическая статистика представляет собой раздел теории вероятностей, направленный на решение обратной задачи: восстановление свойств и природы изучаемого явления (распределения) на основе результатов наблюдений. В отличие от теории вероятностей, где модель считается заданной, в статистике данные экспериментов являются основой для поиска оптимальных статистических решений.

==Опр.== **Вероятностно-статистическая модель** - это математическая конструкция $(X, \mathcal{B}(X), \mathcal{P})$, описывающая условия эксперимента, где 
- $X$ - фазовое пространство наблюдений (обычно $\mathbb{R}$ или $\mathbb{R}^n$),
- $\mathcal{B}(X)$ - борелевская сигма-алгебра над $X$,
- $\mathcal{P}$ - семейство распределений, к которому принадлежит истинное распределение $P$.

==Опр.== **Наблюдение** - случайная величина (или вектор) $X$, значения которой фиксируются в ходе эксперимента; формально считается, что $X$ задана на пространстве $(X, \mathcal{B}(X), P)$ правилом $X(x) = x$.

==Опр.== **Выборка** - это упорядоченная совокупность из $n$ НОРСВ $(X_1, \dots, X_n)$, каждая из которых имеет то же распределение, что и исследуемое наблюдение $X$.

==Опр.== **Параметрическая модель** - модель, в которой семейство распределений $\mathcal{P}$ индексируется *(и явным образом зависит от индекса)* параметром $\theta$ из некоторого множества параметров $\Theta$ (то есть $\mathcal{P} = \{P_\theta, \theta \in \Theta\}$).

==Опр.== **Непараметрические модели** - модель, в которых семейство $\mathcal{P}$ слишком обширно для описания конечным набором параметров *(например, семейство всех непрерывных распределений)*. К основным типам данных, помимо количественных выборок, могут относиться цензурированные данные или данные в шкалах рангов.

==Опр.== **Статистика** - любая измеримая функция от выборки $S(X_1, \dots, X_n)$, не зависящая от неизвестного параметра $\theta$.

==Опр.== **Оценка** - это статистика $T(X_1, \dots, X_n)$, значения которой используются в качестве приближённых значений для неизвестного параметра $\theta$ или некоторой функции от него $\tau(\theta)$.

==Опр.== **Вариационный ряд** - это последовательность элементов выборки, переставленных в порядке неубывания: $X_{(1)} \le X_{(2)} \le \dots \le X_{(n)}$

==Опр.== **Порядковая статистика** - это элемент вариационного ряда $X_{(k)}$, где индекс $k$ называется рангом наблюдения в данном ряду.

Статистики, представляющие собой усредненные значения функций от элементов выборки, называются выборочными характеристиками. К наиболее важным из них относятся:

1) ==Опр.== **Выборочное среднее** - среднее арифметическое элементов выборки: $$\overline{X} = \frac{1}{n} \sum_{i=1}^n X_i$$
2) ==Опр.== **Выборочный момент** ($k$-го порядка) - среднее значение $k$-х степеней элементов выборки: $$\overline{X^k} = \frac{1}{n} \sum_{i=1}^n X_i^k$$
3) ==Опр.== **Выборочная дисперсия** - характеристика разброса выборки, вычисляемая по формуле $$s^2 = \overline{X^2} - (\overline{X})^2 = \frac{1}{n} \sum_{i=1}^n (X_i - \overline{X})^2$$
4) ==Опр.== **Выборочная медиана** - средний элемент вариационного ряда: $$X_{med} = \begin{cases} \frac{1}{2}(X_{(k)} + X_{(k+1)}), & n = 2k \\ X_{(k+1)}, & n = 2k+1 \end{cases}$$

# <h1 id="p1p2"><center>Основные свойства оценок</center></h1>
# <h1 id="p1p2p1"><center>Несмещённые оценки</center></h1>

==Опр.== **Несмещённая оценка $\hat{\theta}_n$** параметрической функции $\tau(\theta)$ - такая оценка, что $$\forall \theta \in \Theta\ :\ \mathbb{E}_{\theta}(\hat{\theta}_n) = \tau(\theta)$$
- Свойство несмещённости гарантирует отсутствие систематической погрешности при многократном повторении серий экспериментов.

==Опр.== **Смещение (бордюр) оценки** $\text{Bias}(\hat{\theta}_n) = \mathbb{E}_\theta (\hat{\theta}_n) - \tau(\theta)$

==Опр.== **Асимптотически несмещённая оценка $\hat{\theta}_n$** параметрической функции $\tau(\theta)$ - такая оценка, что $$\forall \theta \in \Theta\ :\ \lim\limits_{n \to \infty}\mathbb{E}_{\theta}(\hat{\theta}_n) = \tau(\theta)$$
### Теорема о несмещённости выборочного среднего

Пусть $X = (X_1, \dots, X_n)$ - выборка с $\mathbb{E}_\theta X_i = \mu(\theta)$ - равномерно ограниченная на $\Theta$ функция, тогда выборочное среднее $\overline{X}$ - несмещённая оценка $\mu(\theta)$.
#### Доказательство

Используя свойство линейности математического ожидания случайных величин, запишем:
$\mathbb{E}_\theta \overline{X} =$ $\mathbb{E}_\theta \left( \frac{1}{n} \sum\limits_{i=1}^n X_i \right) =$ $\frac{1}{n} \sum\limits_{i=1}^n \mathbb{E}_\theta X_i =$ $\frac{1}{n} \sum\limits_{i=1}^n \mu(\theta) =$ $\frac{1}{n} \cdot n \cdot \mu(\theta)$ $= \mu(\theta)$
Равенство выполняется при всех $\theta \in \Theta$ по определению МО, следовательно, оценка $\overline{X}$ несмещена.

### Теорема о смещении выборочной дисперсии

Пусть $X = (X_1, \dots, X_n)$ - выборка с $\mathbb{E}_\theta X_i = \mu(\theta)$ и $\mathbb{D}_\theta X_i = \sigma^2(\theta)$, где $\mu$ и $\sigma^2$ равномерно ограниченные на $\Theta$ функции, тогда выборочная дисперсия $$s^2 = \frac{1}{n} \sum_{i=1}^n (X_i - \bar{X})^2$$является смещённой оценкой для $\sigma^2(\theta)$, тогда как исправленная выборочная дисперсия $$S^2 = \frac{1}{n-1} \sum_{i=1}^n (X_i - \bar{X})^2$$является несмещённой.

#### Доказательство

Выполним тождественные преобразования суммы квадратов отклонений, вычитая и прибавляя истинное среднее:
$\sum\limits_{i=1}^n (X_i - \bar{X})^2 =$ $\sum\limits_{i=1}^n \left( (X_i - \mu) - (\bar{X} - \mu) \right)^2 =$ $\sum\limits_{i=1}^n \left( (X_i - \mu)^2 - 2(X_i - \mu)(\bar{X} - \mu) + (\bar{X} - \mu)^2 \right) =$ $\sum\limits_{i=1}^n (X_i - \mu)^2 - 2(\bar{X} - \mu) \sum\limits_{i=1}^n (X_i - \mu) + \sum\limits_{i=1}^n (\bar{X} - \mu)^2$
Заметим, что по определению выборочного среднего $\sum\limits_{i=1}^n X_i = n\bar{X}$, откуда $\sum\limits_{i=1}^n (X_i - \mu) = n(\bar{X} - \mu)$. Подставим это:
$\sum\limits_{i=1}^n (X_i - \bar{X})^2 =$ $\sum\limits_{i=1}^n (X_i - \mu)^2 - 2n(\bar{X} - \mu)^2 + n(\bar{X} - \mu)^2 =$ $\sum\limits_{i=1}^n (X_i - \mu)^2 - n(\bar{X} - \mu)^2$

Применим операцию математического ожидания к обеим частям равенства:$$\mathbb{E}_\theta \left( \sum_{i=1}^n (X_i - \bar{X})^2 \right) = \sum_{i=1}^n \mathbb{E}_\theta (X_i - \mu)^2 - n \mathbb{E}_\theta (\bar{X} - \mu)^2$$По определению дисперсии, $\mathbb{E}_\theta (X_i - \mu)^2 = \mathbb{D}_\theta X_i = \sigma^2(\theta)$. При этом:$$\mathbb{E}_\theta (\bar{X} - \mu)^2 =\mathbb{D}_\theta \bar{X} = \mathbb{D}_\theta \left( \frac{1}{n} \sum_{i=1}^n X_i \right) = \frac{1}{n^2} \sum_{i=1}^n \mathbb{D}_\theta X_i = \frac{1}{n^2} \cdot n\sigma^2 = \frac{\sigma^2}{n}$$Подставим найденные значения математических ожиданий:$$\mathbb{E}_\theta \left( \sum_{i=1}^n (X_i - \bar{X})^2 \right) = n\sigma^2 - n \cdot \frac{\sigma^2}{n} = n\sigma^2 - \sigma^2 = (n - 1)\sigma^2$$

Теперь найдём математическое ожидание обычной выборочной дисперсии:
$$\mathbb{E}_\theta S^2 = \mathbb{E}_\theta \left( \frac{1}{n} \sum_{i=1}^n (X_i - \bar{X})^2 \right) = \frac{1}{n} (n - 1)\sigma^2 = \frac{n - 1}{n} \sigma^2$$
Т.к. $\mathbb{E}_\theta [S^2] \neq \sigma^2$, оценка $s^2$ является смещённой. Однако $\frac{n-1}{n}\sigma^2 \xrightarrow[n \to \infty]{} \sigma^2$, то есть она асимптотически несмещена. Для исправленной выборочной дисперсии получаем:$$\mathbb{E}_\theta S^2 = \mathbb{E}_\theta \left( \frac{1}{n-1} \sum_{i=1}^n (X_i - \bar{X})^2 \right) = \frac{1}{n-1} (n-1)\sigma^2 = \sigma^2$$Следовательно, $S^2$ является строго несмещённой оценкой для $\sigma^2$.

# <h1 id="p1p3"><center>Теорема Колмогорова-Смирнова</center></h1>

Пусть имеется выборка $X = (X_1, X_2, \dots, X_n)$ из распределения с неизвестной функцией распределения $F(x, \theta)$.

==Опр.== **Эмпирическая функция распределения (ЭФР)** - статистика вида: $$\hat{F_n}(x) = \frac{1}{n}\sum\limits_{k=1}^n\mathbb{1}_{\{X_k \le x\}}(x)$$
### Свойства ЭФР в фиксированной точке

1. **Распределение:** $$n \hat{F_n}(x) \sim \text{Bin}(n, F(x))$$
2. **Несмещённость:** $$\mathbb{E}\hat{F_n}(x) = F(x)$$
3. **Состоятельность:** По ЗБЧ$$\hat{F_n}(x) \xrightarrow{\mathbb{P}} F(x)$$
4. **Асимптотическая нормальность: По ЦПТ$$\sqrt{n}(\hat{F_n}(x) - F(x)) \xrightarrow{d} \mathcal{N}(0, F(x)(1 - F(x)))$$
### Статистика Колмогорова

==Опр.== **Статистика Колмогорова:** $$D_n = \sup_{\mathbb{R}} |\hat{F_n}(x) - F(x)|$$Где $F$ - функция распределения исследуемого распределения, а $\hat{F_n}$ - Эмпирическая функция распределения.

### Теорема Гливенко-Кантелли (Колмогорова-Смирнова)
$$D_n = \sup_{\mathbb{R}} |\hat{F_n}(x) - F(x)| \xrightarrow[n \to \infty]{a.\forall} 0$$

### Теорема Колмогорова
$$\lim_{n \to \infty} P(\sqrt{n}D_n \le z) = K(z) = \begin{cases} \sum\limits_{k=-\infty}^{\infty} (-1)^k e^{-2k^2 z^2}, & z > 0 \\ \\ 0, & z \le 0 \end{cases}$$

### Двухвыборочная теорема Колмогорова-Смирнова

Пусть имеются две независимые выборки:
- $X = (X_1, \dots, X_n)$ из непрерывного распределения с функцией $F(x)$
- $Y = (Y_1, \dots, Y_m)$ из непрерывного распределения с функцией $G(x)$

Тогда если $\hat{F_n}(x)$ и $\hat{G_m}(x)$ - соответствующие ЭФР, то для **статистики Колмогорова-Смирнова** $$D_{n,m} = \sup_{x \in \mathbb{R}} |F_n(x) - G_m(x)|$$
Верно:  $$\lim_{n,\ m \to \infty} \mathbb{P}\left(\sqrt{\frac{nm}{n+m}} D_{n,m} \le z\right) = K(z)$$

> [!info] **Определение**
> 
> Непрерывная случайная величина $X$ имеет **Гамма-распределение** с параметром формы $\alpha > 0$ и параметром ***интенсивности*** $\beta > 0$, если её плотность вероятности имеет вид: $$p_\Gamma(x) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x} \cdot \mathbb{1}_{(0; +\infty)}(x) \sim \text{Gamma}(\alpha, \beta)$$

_Примечание:_ Существует альтернативная параметризация через параметр ***масштаба*** $\theta = 1/\beta$.

### Конструктивное определение

Если параметр формы $\alpha = k$ является целым числом, то случайная величина $X \sim \text{Gamma}(k, \beta)$ может быть представлена как сумма $k$ НОРСВ имеющих экспоненциальное распределение с параметром $\beta$: $$X = \sum_{i=1}^k E_i, \quad E_i \sim \text{Exp}(\beta)$$
### Числовые характеристики

- Для случайной величины $X \sim \text{Gamma}(\alpha, \beta)$ математическое ожидание и дисперсия равны: $$\mathbb{E}X = \frac{\alpha}{\beta}, \quad \mathbb{D}X = \frac{\alpha}{\beta^2}$$

### Cвойства гамма-распределения

Пусть $X \sim \text{Gamma}(\alpha_1, \beta)$ и $Y \sim \text{Gamma}(\alpha_2, \beta)$ - независимые СВ, тогда:
1. $X + Y \sim \text{Gamma}(\alpha_1 + \alpha_2, \beta)$ - **аддитивность**
2. $\forall c = const\ :\ cX \sim \text{Gamma}\left(\alpha_1, \frac{\beta}{c}\right)$ - **масштабируемость**

## Бета-распределение

> [!info] **Определение**
> 
> Непрерывная случайная величина $X$ имеет **Бета-распределение** на отрезке $[0, 1]$ с параметрами формы $p > 0$ и $q > 0$, если её плотность вероятности имеет вид: $$f_B(x) = \text{B}^{-1}(p, q) x^{p-1} (1-x)^{q-1} \cdot \mathbb{1}_{[0; 1]}(x) \sim \text{Beta}(p, q)$$

### Числовые характеристики

 - Для случайной величины $X \sim \text{Beta}(p, q)$ математическое ожидание и дисперсия имеют вид:$$\mathbb{E}X = \frac{p}{p+q}, \quad \mathbb{D}X = \frac{pq}{(p+q)^2(p+q+1)}$$
### Теорема о преобразовании Гамма-величин

Пусть $Y_1 \sim \text{Gamma}(p, \lambda)$ и $Y_2 \sim \text{Gamma}(q, \lambda)$ - независимые случайные величины с общим параметром интенсивности, тогда случайная величина $X = \frac{Y_1}{Y_1 + Y_2} \sim \text{Beta}(p, q)$.

### Cвойства Бета-распределения

1. $X \sim \text{Beta}(p, q)$ $\Longrightarrow$ $1 - X \sim \text{Beta}(q, p)$ - **симметрия**
2. $\text{Beta}(1, 1) \equiv U_{[0, 1]}$

### Теорема о распределении вариационного ряда бета-распределения

Пусть $U = (U_1, U_2, \dots, U_n)$ - выборка $\sim U[0,1]$. тогда $U_{(k)} \sim \text{Beta}(k, n - k + 1)$

## Распределение Хи-квадрат Пирсона ($\chi^2$)

> [!info] **Конструктивное определение**
> Пусть $Z_1, Z_2, \dots, Z_n$ - НОРСВ, т.ч. $Z_i \sim \mathcal{N}(0, 1)$, тогда случайная величина: $$X = \sum_{i=1}^n Z_i^2 \sim \chi^2_n$$ имеет **распределение Хи-квадрат с $n$ степенями свободы**.

### Плотность распределения

Распределение абсолютно непрерывно, его плотность вероятности отлична от нуля только при $x > 0$ и задается формулой:$$p_{\chi^2_n}(x) = \frac{x^{n/2 - 1} e^{-x/2}}{2^{n/2} \Gamma(n/2)} \cdot \mathbb{1}_{[0; +\infty)}(x)$$
### Числовые характеристики

- Для случайной величины $X \sim \chi^2_n$ математическое ожидание и дисперсия равны:$$\mathbb{E}X = n, \quad \mathbb{D}X = 2n$$
### Cвойства распределения Хи-квадрат

Пусть $X \sim \chi^2_n$ и $Y \sim \chi^2_m$ - независимые случайные величины, тогда:
1. $X + Y \sim \chi^2_{n+m}$ - **аддитивность**
2. $\chi^2_n \equiv \text{Gamma}\left(\frac{n}{2}, \frac{1}{2}\right)$

## Распределение Стьюдента

> [!info] **Конструктивное определение**
> 
> Пусть случайная величина $Z \sim \mathcal{N}(0, 1)$, а случайная величина $X \sim \chi^2_n$, причём они независимы, тогда случайная величина: $$T = \frac{Z}{\sqrt{X/n}} \sim Student(n)$$имеет **распределение Стьюдента с $n$ степенями свободы**.

### Плотность распределения

$$p_{Student}(y) = \frac{\Gamma((n+1)/2)}{\sqrt{\pi n} \Gamma(n/2)} \left(1 + \frac{y^2}{n}\right)^{-\frac{n+1}{2}}$$

### Свойства распределения Стьюдента

Пусть $T \sim Student(n)$, тогда:
1. $T \sim -T$ - **симметричность**
2. $Student(1) \equiv C(0)$
3. $\forall n > 1\ :\ \mathbb{E}T = 0$
4. $\forall n > 2\ :\ \mathbb{D}[T] = \frac{n}{n-2}$

### Теорема о распределения Стьюдента
$$T \sim Student(n) \quad \Longrightarrow \quad T \xrightarrow[n \to \infty]{d} \mathcal{N}(0, 1)$$

##  Распределение Фишера

> [!info] **Конструктивное определение**
> Пусть $X_1 \sim \chi^2_{n_1}$ и $X_2 \sim \chi^2_{n_2}$ - независимые случайные величины, тогда случайная величина: $$Y = \frac{X_1 / n_1}{X_2 / n_2} \sim F_{n_1, n_2}$$называется **распределением Фишера (или F-распределением) со степенями свободы $n_1$ и $n_2$**.

### Плотность распределения
$$p_F(x) = \frac{\Gamma(\frac{n+m}{2})}{\Gamma(\frac{n}{2})\Gamma(\frac{m}{2})} \left(\frac{n}{m}\right)^{n/2} \frac{x^{n/2 - 1}}{\left(1 + \frac{n}{m}x\right)^{\frac{n+m}{2}}} \cdot \mathbb{1}_{(0; +\infty)}(x)$$

### Числовые характеристики

- Математическое ожидание случайной величины $Y \sim F_{n_1,n_2}$ равно: $\mathbb{E}[F] = \frac{n_2}{n_2-2}$
- Дисперсия распределения Фишера имеет вид $\mathbb{D}Y = \frac{2n_2^2(n_1+n_2-2)}{n_1(n_2-2)^2(n_2-4)}$ и определена только для $n_2 > 4$.

### Свойства F-распределения

1. $Y \sim F_{n, m}$ $\Rightarrow$ $Y^{-1} \sim F_{m, n}$ - **обратимость**
2. $T \sim Student(n)$ $\Rightarrow$ $T^2 \sim F_{1, n}$

# <h1 id="p2p1"><center>Регулярные статистики</center></h1>

==Опр.== **Вклад выборки** - $U(X, \theta) = \frac{\partial}{\partial \theta} \ln L(X, \theta)$, где $L$ - многомерная плотность выборки $X$.

==Опр.== **Информация Фишера** *(выборки $X$)* - $I(\theta) = \mathbb{E}_\theta (U(X, \theta))^2$

==Опр.== **Информация Фишера** *(наблюдения $X_1$ с плотностью $p$)* - $i(\theta) = \mathbb{E}_\theta (\frac{\partial}{\partial \theta} \ln p(X_1, \theta))^2$

==Опр.== Пусть $\hat\theta$ - оценка параметра $\theta$ (из распределения с плотностью $p$), который принимает множество значений $\Theta$. Для $\hat\theta$ **выполнены условия регулярности**, если:
1) $\Theta$ - открытая связная область
2) $N_p := \{x\ |\ p(x, \theta) > 0\}$ не зависит от $\theta$
3) $\forall x \in N_p\ :\ \exists \frac{\partial}{\partial \theta}\ln p(x, \theta)$
4) $\frac{\partial}{\partial \theta}\int\limits_{N_p}p(x, \theta)dx = \int\limits_{N_p}\frac{\partial}{\partial \theta}p(x, \theta)dx$
5) $\frac{\partial}{\partial \theta}\int\limits_{N_p}\hat\theta\ p(x, \theta)dx = \int\limits_{N_p}\hat\theta\frac{\partial}{\partial \theta}p(x, \theta)dx$
6) $\exists I(\theta) > 0$ - информация Фишера.

### Соотношение информации Фишера выборки и наблюдения
$$I(\theta) = ni(\theta)$$

### Эквивалентные определения информации Фишера
$$i(\theta) = \mathbb{E}_\theta \left(\frac{\partial}{\partial \theta} \ln p(X_1, \theta)\right)^2 = -\mathbb{E}_\theta \left(\frac{\partial^2}{\partial \theta^2} \ln p(X_1, \theta)\right)$$

### Вклад регулярной выборки

Если для семейства распределений выполнены условия регулярности, то $\mathbb{E}_\theta U(X, \theta) = 0$

# <h1 id="p2p2"><center>Метод моментов</center></h1>

==Опр.== **Метод моментов** - метод нахождения статистики, которая хорошо приближает оцениваемый параметр в параметрической статистике.

> [!tip] **Суть метода**
> Метод моментов основан на простой эвристике: теоретические моменты распределения должны быть близки к своим эмпирическим аналогам в силу ЗБЧ. Если распределение зависит от $k$ неизвестных параметров, то составляется система, в которых теоретические моменты приравниваются к выборочным. Решение этой системы относительно параметров и принимается за оценку метода моментов (ОММ).

### Алгоритм

Пусть $\theta = (\theta_1, \dots, \theta_k) \in \Theta \subset \mathbb{R}^k$ - исследуемый вектор-параметр.

1. Вычисляются первые $k$ теоретических начальных моментов как функции от неизвестных параметров:$$\alpha_m(\theta) = \mathbb{E}_\theta X_1^m, \quad m \in \overline{1, k}$$
2. На основе выборки рассчитываются первые $k$ выборочных начальных моментов:$$a_m = \frac{1}{n} \sum_{i=1}^n X_i^m, \quad m \in \overline{1, k}$$
3. Составляется система уравнений метода моментов:$$ 
    \begin{cases}
    \alpha_1(\theta_1, \dots, \theta_k) = a_1 \
    \alpha_2(\theta_1, \dots, \theta_k) = a_2 \
    \dots \
    \alpha_k(\theta_1, \dots, \theta_k) = a_k
    \end{cases}
    $$
4. Решением данной системы является вектор $\hat{\theta}_n = (\hat{\theta}_{1n}, \dots, \hat{\theta}_{kn})^T$, который и называется ОММ.

### Теорема о состоятельности ОММ

Пусть $X = (X_1, \dots, X_n)$  - выборка, у которой $\forall i \in \overline{1, k}\ :\ \exists\mathbb{E}_\theta |X_1|^i < \infty$. Пусть $a = (\mathbb{E}_\theta X_1^1, \dots, \mathbb{E}_\theta X_1^k)^T$. Если функции $\alpha_m(\theta)$ непрерывна на $\Theta$, а отображение $\alpha(\theta) = (\alpha_1(\theta), \dots, \alpha_k(\theta))^T$ диффеоморфно в окрестности точки $a$, то $\hat{\theta}_{ОММ}$ является слабо состоятельной оценкой параметра $\theta$.

# <h1 id="p2p3"><center>Метод максимального правдоподобия</center></h1>

==Опр.== **Метод максимального правдоподобия** - метод нахождения статистики, которая хорошо приближает оцениваемый параметр в параметрической статистике.

> [!tip] **Суть метода**
> Метод максимального правдоподобия ***(ОМП - Оценка Максимального Правдоподобия)***, опирается на принцип: в качестве оценки параметра выбирается то значение $\theta$, которое максимизирует вероятность получения именно той выборки, которая была фактически наблюдена.

Пусть $X = (X_1, \dots, X_n)$ - выборка с плотностью $X_i$ равной $p(x, \theta)$.

==Опр.== **Функция правдоподобия** по выборке $L(X, \theta) = \prod\limits_{k=1}^n p(X_k, \theta)$

==Опр.== **Логарифмическая функция правдоподобия** по выборке $\mathcal{L}(X, \theta) = \ln L(X, \theta) = \sum\limits_{k=1}^n \ln p(X_k, \theta)$

==Опр.== $\hat{\theta}_{ОМП} = \arg\max\limits_{\theta \in \Theta}\mathcal{L}(X, \theta)$

### Алгоритм

- Ищем глобальный максимум функции, используя инструменты математического анализа *(например, метод множителей Лагранжа)*.

### Свойства регулярной ОМП

Если для $\hat{\theta}_{ОМП}$ выполняется условие регулярности, то она:
1) Слабо состоятельна
2) Асимптотически нормальна

### Теорема о предельном распределении для нерегулярной ОМП

Для выборки из распределения $U[0, \theta]$ СВ $n(\theta - X_{(n)})$ слабо сходится к экспоненциальному закону распределения с параметром интенсивности $1/\theta$.

# <h1 id="p2p4p1"><center>Эффективные оценки</center></h1>

==Опр.== **Функция риска (среднеквадратичная ошибка)** оценки $\hat\theta_n$ параметра $\theta$ - $R_n(\theta) = \mathbb{E}_{\theta}(\hat\theta_n - \theta)^2$

==Опр.== **Оптимальная оценка** - оценка $\theta^*$, для которой верно: $\forall \theta\ :\ \theta^* = \arg \min\limits_{\hat\theta} R_n(\theta)$

==Опр.== **Эффективная оценка** - оптимальная несмещённая оценка $\theta^*$, для которой верно $\forall \theta \in \Theta\ \forall \hat\theta$ - оценка $\theta$ $:\ \mathbb{D}_{\theta}\theta^* \le \mathbb{D}_{\theta}\hat\theta$

### Теорема о разложении риска
$$R_n(\theta) = \mathbb{D}_{\theta}\hat\theta_n + Bias^2(\hat\theta_n)$$

# <h1 id="p2p4p2"><center>Неравенство Рао-Крамера</center></h1>

## Неравенство Рао-Крамера

Пусть для оценки $\theta^*$ параметрической функции $\tau(\theta)$ выполняются условия регулярности, тогда $$\forall \theta \in \Theta\ :\ \mathbb{D}_{\theta}\theta^* \ge \frac{(\tau'(\theta))^2}{I(\theta)}$$

## Критерий эффективности по Рао-Крамеру

Если $\theta^*$ удовлетворяет условию неравенства Рао-Крамера и $\forall \theta \in \Theta\ :\ \mathbb{D}_{\theta}\theta^* = \frac{(\tau'(\theta))^2}{I(\theta)}$, то $\theta^*$ - эффективная оценка.

---

# <h1 id="p2p4p3"><center>Экспоненциальное семейство распределений</center></h1>

==Опр.== **Семейство экспоненциальных распределений** - распределения, чьи плотности можно представить в виде $p(x, \theta) = \exp(A(\theta)B(x) + C(x) + D(\theta))$, где $A,\ B,\ C$ и $D$ - измеримые функции.

==Опр.== **Экспоненциальное семейство особого типа** относительно статистики $\theta^*$ - часть многомерного экспоненциального семейства, в котором $B(x) = \theta^*$.

### Примеры распределений из экспоненциального семейства

- Нормальное
- Пуассоновское
- Биномиальное
- Бернуллиевское
- Экспоненциальное
- и др.

### Критерий эффективности через экспоненциальное семейство

Если $\theta^*$ удовлетворяет условию регулярности и выборка $X$ из экспоненциального семейства особого типа относительно $\theta^*$, то $\theta^*$ - эффективная оценка параметра $\theta$.

---

# <h1 id="p2p4p4"><center>Критерий эффективности</center></h1>

Если оценка $\theta^*$ параметрической функции $\tau(\theta)$ удовлетворяет условию регулярности и $$\exists a(\theta)\ :\  \theta^* - \tau(\theta) \stackrel{a.\forall}= a(\theta)U(X, \theta)$$то $\theta^*$ - эффективная оценка и при этом $$a(\theta) \stackrel{a.\forall}= \frac{\tau'(\theta)}{I(\theta)}$$

# <h1 id="p2p4p5"><center>Достаточные и полные статистики</center></h1>

==Опр.== **Достаточная статистика $T$** для параметра $\theta$ - такая статистика, если $\exists f(t)\ \forall B \in \mathcal{B}(\mathbb{R}^n)\ \forall t\ :\ P_\theta(X \in B|T(X) = t) = f(t)$ и не зависит от $\theta$

==Опр.== **Полная статистика $T$** для параметра $\theta$ - такая статистика, если $\forall \varphi \in \mathcal{M}_d\ :\ \mathbb{E}_\theta \varphi(T) = 0 \Leftrightarrow \varphi \stackrel{a.\forall}\equiv 0$

# <h1 id="p2p4p5p1"><center>Критерий факторизации</center></h1>

Статистика $T(X)$ является достаточной для параметра $\theta$ $\Longleftrightarrow$ $\exists g, h\ (\in \mathcal{M}) > 0\ :$ $L(X, \theta) = g(T(X), \theta) \cdot h(X)$

---

# <h1 id="p2p4p5p2"><center>Теорема Рао-Блэкуэлла-Колмогорова</center></h1>

Пусть $\hat{\theta}$ - произвольная несмещённая оценка параметрической функции $\tau(\theta)$, а $T(X)$ - достаточная статистика, тогда оценка $\tilde{\theta} = \mathbb{E}(\hat{\theta} \mid T)$ обладает следующими свойствами:

1. Она является корректной статистикой.
2. Она является несмещённой оценкой для $\tau(\theta)$.
3. $\mathbb{D}_\theta \tilde{\theta} \le \mathbb{D}_\theta \hat{\theta}$.

---

# <h1 id="p2p4p5p3"><center>Теорема Лемана-Шеффе</center></h1>

### Теорема Лемана-Шеффе

Если достаточная статистика $T(X)$ является полной, и на её основе найдена некоторая несмещённая оценка $\tilde{\theta} = f(T)$ для параметрической функции $\tau(\theta)$, то эта оценка является единственной эффективной и при этом несмещённой оценкой.

### Алгоритм построения оптимальной оценки

1. Найти функцию правдоподобия $L(X, \theta)$ и с помощью критерия факторизации выделить достаточную статистику $T(X)$.
2. Проверить статистику $T$ на полноту.
3. Найти любую несмещённую оценку $\hat{\theta}$.
4. Вычислить условное математическое ожидание $\tilde{\theta} = \mathbb{E}(\hat{\theta} \mid T)$. Полученный результат гарантированно будет единственной эффективной оценкой.
