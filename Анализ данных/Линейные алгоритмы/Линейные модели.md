==Опр.== Линейная модель имеет вид **дискриминантной функции**:$$f(x) = w_0 + \sum\limits_{i=1}^d w_i x_i = w_0 + \langle w, x \rangle$$
- $w = (w_1, \dots, w_d)^T \in \mathbb{R}^d$ - веса линейной модели
- $w_0 \in \mathbb{R}$ - **порог** (свободный, корректирующий, член линейной модели)
- Для упрощения записи часто вводят фиктивный признак $x_0 = 1$, тогда:$$f(x) = \langle w, x \rangle = w^T x$$где $w$ и $x$ - это уже расширенные векторы, включающие $w_0$ и $x_0$.

==Опр.== **Обучение линейной модели** - поиск вектора весов $w$, минимизирующего сумму потерь на обучающей выборке.

### Линейная [[Типы задач машинного обучения|классификация]]

==Опр.== **Линейный классификатор** - это алгоритм $a(x)$, значение которого определяется знаком функции линейной модели:$$a(x) = \text{sign}(f(x)) = \begin{cases} +1, &\text{если } f(x) > 0 \\ -1, &\text{если } f(x) \le 0 \end{cases}$$
- То есть в линейной классификации $Y = \{\pm 1\} \cong B$

> [!tip] Геометрическая интерпретация
> Уравнение $\langle w, x \rangle + w_0 = 0$ задает **разделяющую гиперплоскость** в $d$-мерном пространстве признаков. Вектор весов $w$ является нормалью к этой гиперплоскости. Скалярное произведение $\langle w, x \rangle$ пропорционально расстоянию от точки $x$ до плоскости со знаком. Свободный член $w_0$ определяет смещение плоскости относительно начала координат.

### Линейная [[Типы задач машинного обучения|регрессия]]

Линейная регрессия решает задачу восстановления непрерывной зависимости. То есть $Y \subseteq \mathbb{R}$

==Опр.== **Линейная регрессия** - модель вида:
$$a(x) = \langle w, x \rangle + w_0 \quad (\text{или можно записать просто}\ \langle w, x \rangle)$$

#### Аналитическое решение (для квадратичной функции потерь)

Если добавить фиктивный признак $x_0=1$ и использовать квадратичную функцию потерь, можно получить решение в замкнутой форме. Пусть $F$ - матрица "объекты-признаки" размера $\ell \times (d+1)$, $y$ - вектор целевых значений, тогда:$$Q(w) = (Fw - y)^T(Fw - y)$$Найдём стационарные точки:$$\nabla Q(w) = \frac{\partial Q}{\partial w}(w)=2F^T(Fw-y) = 0\ \Longleftrightarrow\ w = (F^TF)^{-1}F^Ty$$Проверим что это точка глобального минимума: $$\frac{\partial}{\partial w}\nabla Q(w) = 2 F^TF \ge 0\ -\ \text{в силу свойств квадратичной формы ЕП}$$Значит $w = (F^TF)^{-1}F^Ty$ - действительно оптимальные веса при квадратичной функции потерь.

### +ы & -ы линейных моделей

> [!success] Преимущества
> *   Простота реализации и применения.
> *   Высокая скорость обучения и работы.
> *   Хорошо работают при большом количестве признаков.
> *   Устойчивы при малом количестве данных (благодаря регуляризации).
> *   Интерпретируемость (веса показывают важность признаков).

> [!failure] Недостатки
> *   Предполагают линейную зависимость, что может быть слишком грубым допущением.
> *   Чувствительны к масштабу признаков (требуют нормализации/стандартизации).
