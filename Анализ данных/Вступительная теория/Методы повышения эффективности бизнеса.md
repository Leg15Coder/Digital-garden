> [!info] Общая постановка задачи
> Классическая задача бизнеса - максимизация прибыли. Формально, прибыль $\Pi$ можно представить как разность между доходами $R$ и расходами $C$:
> $$\Pi = R - C \rightarrow \max$$
> Методы, представленные ниже, направлены на увеличение компоненты $R$ (доходы) и уменьшение компоненты $C$ (расходы/потери).

### 1. Увеличение доходной части ($R \uparrow$)

==Опр.== **Лидогенерация** - это процесс привлечения потенциальных клиентов (лидов), заинтересованных в продукте или услуге. Является первым этапом воронки продаж.

==Опр.== **Рекомендательные системы** - это подкласс методов фильтрации информации, предназначенный для предсказания "рейтинга" или "предпочтения", которое пользователь присвоит еще не купленному товару.

==Опр.== **Умное ценообразование (Dynamic pricing)** - это стратегия установления гибких цен на товары или услуги в зависимости от текущих рыночных условий, спроса, предложения и поведения клиента.

> [!example] Пример взаимосвязи
> 1.  **Лидогенерация** доставляет "сырой" трафик.
> 2.  **Рекомендательная система** увеличивает конверсию этого трафика (вероятность покупки $P(purchase|user)$), предлагая релевантный товар.
> 3.  **Умное ценообразование** максимизирует маржинальность каждой совершенной покупки, подбирая оптимальную цену $p_{opt}$.

### 2. Минимизация потерь (Убытки $\downarrow$)

==Опр.== **Рисковый скоринг (Credit Scoring)** - это статистическая модель, используемая для оценки вероятности дефолта (невозврата) заемщика.

==Опр.== **Антифрод (Fraud Detection)** - это комплекс мероприятий и алгоритмов, направленных на выявление и предотвращение мошеннических транзакций.

### 3. Оптимизация затрат ($C \downarrow$)

==Опр.== **Оптимизация расходов (Cost Optimization)** - это процесс минимизации затрат на бизнес-процессы без потери качества. Включает в себя:
1.  Снижение переменных издержек.
2.  Сокращение постоянных издержек.
3.  Повышение эффективности использования ресурсов.

==Опр.== **Автоматизация/Роботизация процессов (RPA)** - замена ручного труда сотрудников программными роботами для выполнения рутинных, повторяющихся задач с фиксированными правилами.

### Итоговая структурная схема

| Направление            | Метод                       | Целевая функция (упрощенно)                 |
| :--------------------- | :-------------------------- | :------------------------------------------ |
| **Увеличение доходов** | Лидогенерация               | Увеличение мощности множества клиентов      |
|                        | Рекомендательные системы    | Увеличение $P(purchase)$                    |
|                        | Умное ценообразование       | Увеличение $p_{opt}$ (оптимальной цены)     |
| **Снижение потерь**    | Рисковый скоринг            | Снижение $P(debt)$                          |
|                        | Антифрод                    | Снижение доли мошеннических транзакций      |
| **Оптимизация затрат** | Оптимизация расходов        | Снижение постоянных/переменных издержек     |
|                        | Замена сотрудников роботами | Снижение операционных затрат на ручной труд |
