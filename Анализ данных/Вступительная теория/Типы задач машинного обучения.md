> [!tip] Стандарт классификации
> В [[Формализация понятий Data Science|машинном обучении]] все задачи классифицируются по двум основным признакам: типу выходной переменной (что прогнозируем) и наличию обучающей разметки (наличию целевых значений $y$).

### 1. Базовая классификация по характеру обучения

==Опр.== **Обучение с учителем** - парадигма машинного обучения, в которой модель обучается на размеченной выборке $D = \{(x_i, y_i)\}_{i=1}^n$. Для каждого объекта $x_i$ (признаковое описание) известен правильный ответ $y_i$ (целевая переменная). Задача - аппроксимировать функцию $f: X \rightarrow Y$.

==Опр.== **Обучение без учителя** - парадигма машинного обучения, в которой модель обучается на неразмеченной выборке $D = \{x_i\}_{i=1}^n$. Задача - обнаружить скрытые закономерности, структуру или зависимости в данных без использования "правильных ответов".

### 2. Задачи обучения с учителем

В рамках обучения с учителем выделяют два основных типа задач, которые определяются природой целевой переменной $Y$.

#### 2.1. Классификация

==Опр.== **Классификация** - это задача отображения входного объекта $x \in X$ в дискретную целевую переменную $y \in \{C_i\}_{i=1}^k$, где $C_i$ - метки классов.

Модель ищет решающую границу (разделяющую поверхность), которая отделяет объекты одного класса от другого в пространстве признаков.

> [!example] Бизнес-примеры задач классификации
> 1.  **Кредитный скоринг:** $y \in \{\text{вернет кредит}, \text{не вернет кредит}\}$.
> 2.  **Сентимент-анализ (тональность текста):** $y \in \{\text{позитивный}, \text{негативный}, \text{нейтральный}\}$.
> 3.  **Прогнозирование оттока:** $y \in \{\text{клиент уйдет}, \text{клиент останется}\}$.
> 4.  **Тематическое моделирование (с учителем):** $y \in \{\text{спорт}, \text{политика}, \text{экономика}, ...\}$, где набор тем задан экспертом заранее.

#### 2.2. Регрессия

==Опр.== **Регрессия** - это задача отображения входного объекта $x \in X$ в непрерывную целевую переменную $y \in \mathbb{R}$.

Цель - построить функцию, описывающую зависимость между переменными. Простейший вид - линейная регрессия:
$$y = \beta_0 + \beta_1 x_1 + ... + \beta_n x_n + \epsilon$$
где $\beta$ - коэффициенты модели, $\epsilon$ - ошибка.

> [!example] Бизнес-примеры задач регрессии
> 1.  **Прогнозирование финансовых метрик:** размер выплаты по задолженности, цена ценной бумаги.
> 2.  **Производственные задачи:** прогнозирование массы дефекта на производстве.
> 3.  **Физические величины:** прогнозирование температуры воздуха, давления, скорости ветра.

### 3. Задачи обучения без учителя

В рамках обучения без учителя центральное место занимает задача кластеризации.

#### 3.1. Кластеризация

==Опр.== **Кластеризация** - это задача разбиения множества объектов $X = \{x_1, x_2, ..., x_n\}$ на группы (кластеры) таким образом, чтобы объекты внутри одной группы были максимально похожи друг на друга (высокая внутрикластерная гомогенность), а объекты из разных групп - максимально различны (высокая межкластерная гетерогенность).

$$\text{Найти разбиение: } X = \bigcup_{j=1}^{k} C_j, \quad C_i \cap C_{j \neq i} = \emptyset$$

Количество кластеров $k$ может быть задано априори или определено алгоритмом в процессе работы.
