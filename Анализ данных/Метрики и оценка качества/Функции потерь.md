## Функции потерь в задачах классификации
### Логистическая (LogLoss / Binary Cross-Entropy)

$$L(M) = \ln(1 + e^{-M})$$
#### Особенности
1) Используется в логистической регрессии. 
2) Сильно штрафует за уверенные ошибки.

> [!important] Интерпретация
> Выходы модели logloss - вероятности принадлежности к классу 1.

#### Доказательство

1. Рассмотрим константный прогноз вероятности $P(y_i = 1) = p$ для выборки из $n$ объектов:$$Q(p) = -\sum [y_i \ln p + (1-y_i) \ln(1-p)] \to \min_{p}$$
2. Берем производную по $p$ *(ищем глобальный минимум)*:$$Q'(p) = - \left( \frac{\sum y_i}{p} - \frac{\sum(1-y_i)}{1-p} \right) = 0$$
3. Пусть доля единиц в выборке $\alpha = \frac{\sum y_i}{n}$, тогда:$$\frac{\alpha n}{p} - \frac{n - \alpha n}{1 - p} = 0 \implies \frac{\alpha}{p} = \frac{1-\alpha}{1-p}$$$$\alpha(1-p) = p(1-\alpha) \implies \alpha - \alpha p = p - \alpha p \implies p = \alpha$$
Таким образом, оптимальный прогноз для LogLoss - это доля положительного класса в выборке ***(эмпирическая вероятность)***.

### Кросс-энтропия (Categorical Cross-Entropy)

$$softmax(k, x) = \frac{e^{x_k}}{\sum\limits_{i=1}^{K}e^{x_i}}$$ Softmax - функция, которая переводит выходные значения модели в вероятности пропорционально степени уверенности. Функция кросс-энтропии выглядит так:$$L_f(x, y) = -\sum_{j=1}^{K} y_j \ln(softmax(j, f(x)))$$
#### Особенности
1) Обобщение LogLoss на $K$ классов.
2) Обычно используется в нейронных сетях.

### Шарнирная (Hinge Loss)

$$L(M) = \max(0, 1 - M)$$
#### Особенности
1) Применяется в методе опорных векторов. 
2) Штрафует даже за верные ответы, если объект находится слишком близко к разделяющей границе ($M < 1$).

### Экспоненциальная (Exponential Loss)
   $$L(M) = e^{-M}$$
#### Особенности
1) Используется в алгоритме AdaBoost. 
2) Чрезвычайно чувствителен к выбросам.

## Функции потерь в задачах регрессии
### MSE (Mean Squared Error)
   $$L_f(x, y) = (y - f(x))^2$$
#### Особенности
1) Чувствительна к выбросам из-за возведения ошибки в квадрат.

> [!important] Интерпретация
> MSE оптимально ищет среднее значение целевого параметра.

#### Доказательство
Нужно найти такое $c$, которое минимизирует функционал ошибки:$$Q(c) = \sum_{i=1}^{n} (y_i - c)^2 \to \min_{c}$$
1. Возьмем производную по $c$ *(стационарная точка)*:$$Q'(c) = \sum_{i=1}^{n} 2(y_i - c) \cdot (-1) = -2 \sum_{i=1}^{n} (y_i - c)$$
2. Приравняем производную к нулю:$$\sum_{i=1}^{n} y_i - \sum_{i=1}^{n} c = 0 \implies \sum y_i - n \cdot c = 0$$
3. Разрешим относительно $c$:$$c = \frac{1}{n} \sum_{i=1}^{n} y_i = \bar{y}$$
Таким образом, оптимальный прогноз для MSE - это среднее арифметическое.

### MAE (Mean Absolute Error)
   $$L_f(x, y) = |y - f(x)|$$
#### Особенности
   1) Устойчива к выбросам (робастна).

> [!important] Интерпретация
> MAE оптимально ищет медиану целевого параметра.

#### Доказательство
Минимизируем сумму абсолютных отклонений:$$Q(c) = \sum_{i=1}^{n} |y_i - c| \to \min_{c}$$Есть проблема: $Q$ дифференцируема в $n$ точках. Воспользуемся методом матожиданий. Пусть $y$ имеет распределение с функцией $F$ и конечным первым моментом.

1) Предположим, что $F$ непрерывна в точке $c$.
	1) Пусть $g(c) = \mathbb{E}|y - c|$ $= \int\limits_{\mathbb{R}}|y - c| dF(y)$ - она выпуклая и дифференцируема всюду. Подынтегральное выражение имеет производную по $c$ почти всюду.
	2) $g'(c) =$ $\int\limits_{\mathbb{R}}sign(y - c) dF(y) =$ $-(\int\limits_{c}^{+\infty}dF(y) - \int\limits_{-\infty}^cdF(y)) =$ $F(c + 0) + F(c - 0) - 1 =$ $2F(c) - 1$
	3) Стационарная точка $c$ имеет свойство: $F(c) = \frac{1}{2}$. Значит ровно половина значений - меньше, и ровна половина - больше $c$, т.е. $c$ - медиана.
2) Рассмотрим случай атомарности распределения в точке $c$.
	1) Пусть $F_n(c) = \frac{1}{n}\sum\limits_{k=1}^n\mathbb{1}_{\{x_k \le c\}}$
	2) $g(c) = \partial F_n(c) = [2F(c-0) - 1; 2F(c+0)-1]$ - субградиент $F_n$ по $c$
	3) Условие стационарности: $F_n(c-0) \le \frac{1}{2} \le F_n(c)$ $\Rightarrow$ $c$ - произвольная точка между центральными порядковыми статистиками, то есть медиана в классическом смысле.

### MAPE (Mean Absolute Percentage Error)

$$MAPE = \frac{1}{n} \sum \frac{|y_i - f(x_i)|}{|y_i|}$$
#### Особенности
1) Оценивает ошибку в процентах относительно истинного значения.

> [!important] Интерпретация
> MAPE оптимально ищет взвешенную медиану целевого параметра, где вес определяется обратно пропорционально целевому значению.
