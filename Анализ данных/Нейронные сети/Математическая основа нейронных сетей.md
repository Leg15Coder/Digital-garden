### Математическая модель нейрона

==Опр.== **(Искусственный) Нейрон** - это [[Линейные модели|линейная модель]], за которой следует нелинейная функция активации. Нейрон описывается формулой$$u(x) = f(\langle w, x \rangle + b)$$Где:
- $x = (x_1, \dots, x_n)$ - входные данные (вектор признаков)
- $\Sigma = \sum\limits_{i=1}^{n} w_i x_i + b = \langle w, x \rangle + b$ - взвешенная сумма признаков (линейная модель)
- $f$ - **нелинейная функция активации**.

## Теорема Цыбенко (Универсальная теорема аппроксимации, 1989)

> [!important] Универсальная теорема аппроксимации
> Нейронная сеть даже с одним скрытым слоем при достаточном количестве нейронов способна приблизить любую непрерывную функцию на компакте с любой наперед заданной точностью.

## Многослойная сеть

Один нейрон - это функция многих переменных ($\mathbb{R}^n \to \mathbb{R}$), а слой нейросети - вектор из нейронов, то есть функция вида $\mathbb{R}^n \to \mathbb{R}^d$. Если $\vec{f}$ - вектор-функция многих переменных активации, $W$ - тензор весов, а $b$ - вектор смещения, то слой нейросети можно описать формулой:$$a(x) = \vec{f}(Wx + b)$$
Сеть состоит из последовательности слоёв:
1. **Входной слой:** Принимает признаки.
2. **Скрытые слои:** Вычисляют промежуточные представления.
3. **Выходной слой:** Выдаёт итоговый прогноз (числа или вероятности).

### Функции активации

1) **Sigmoid:** $$\sigma(z) = \frac{1}{1 + e^{-z}}$$Диапазон $(0, 1)$. Имеет затухание градиентов при больших значениях $|z|$.

2) **Tanh:** $$tanh(z) = \frac{e^z - e^{-z}}{e^z + e^{-z}}$$Диапазон $(-1, 1)$. Центрирована относительно нуля.

3) **ReLU:** $$ReLU(z) = \max(0, z)$$Диапазон $\mathbb{R}$. Самая популярная благодаря простоте и отсутствию затухания градиента в положительной области. НО: имеет «умирающие нейроны» при $z < 0$.

4) **Leaky ReLU:** $$LeaReLU(z) = \max(\alpha z, z)$$Улучшенный ReLU, который решает проблему «смерти нейронов».

5) **Softmax:**$$\sigma(z)_i = \frac{e^{z_i}}{\sum\limits_{j=1}^{K} e^{z_j}}$$Используется на выходе для классификации, чтобы превратить выходные значения в распределение вероятностей (сумма всегда = 1).
