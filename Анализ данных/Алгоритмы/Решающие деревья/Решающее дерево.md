==Опр.== **Решающее дерево** - это непараметрический [[Типы задач машинного обучения|алгоритм обучения]], который представляет собой иерархическую структуру (граф).

### Структура решающего дерева

1) **Внутренние узлы:** содержат условия (предикаты) на элементы выборки.

Предикат в узле имеет вид $\{x_i \in R_j\}$, где $R_j$ - некоторая область значений признаков $x$. Если предикат равен истине, то ответ для данного элемента выборки следует искать в $j$-м поддереве, иначе проверять следующий предикат.

2) **Листья:** содержат прогноз (класс для классификации или число для регрессии).

Когда алгоритм доходит до узла, он должен получить ответ. В листе всегда однозначно определён ответ для данного элемента выборки.

## Алгоритм построения решающего дерева (жадный)

Дерево строится путем рекурсивного разбиения выборки:
1.  Выбирается признак (пусть его номер будет $j$) и порог $t$.
2.  Выборка $Q$ делится на две части: $L$ (где $x^{(j)} \le t$) и $R$ (где $x^{(j)} > t$).
3.  Процесс повторяется для полученных узлов, пока не будет достигнут критерий остановки.

### Критерии информативности $H(R)$

Выбор $j$ и $t$ на 1-м шаге алгоритма происходит так, чтобы минимизировать функционал качества разбиения:
$$G(j, t) = \frac{|L|}{|Q|} H(L) + \frac{|R|}{|Q|} H(R) \to \min_{j, t}$$

Где $H(R)$ - мера «неоднородности» подмножества объектов в узле:

#### Для классификации (на $K$ классов):

1. Критерий Джини: $$H(R) = \sum_{k=1}^{K} p_k (1 - p_k) = 1 - \sum_{k=1}^{K} p_k^2$$
2. Энтропийный критерий: $$H(R) = - \sum_{k=1}^{K} p_k \ln p_k$$
3. Критерий ошибки классификации: $$H(R) = 1 - \max(p_k)$$

#### Для регрессии:

* **Среднеквадратичная ошибка (MSE):** $$H(R) = \frac{1}{|R|} \sum_{x_i \in R} (y_i - \bar{y})^2$$где $\bar{y}$ - среднее значение целевого параметра в узле


### Критерий прогноза листа

Есть различные способы выбора критерия ответа для листа, самые популярные:

- **Для классификации**: ответ в листе - самый частый класс в нём
- **Для регрессии**: среднее значение целевого параметра у элементов в листе.

### Обрезание деревьев

С увеличением глубины дерева увеличивается переобучение данного дерева. Для борьбы с подобным переобучением ограничивают глубину дерева.

==Опр.== **Обрезание дерева** - ограничение его глубины.

==Опр.== **Пре-обрезание дерева** - обрезание дерева во время его построения (слишком большая глубина - пора остановится).

==Опр.== **Пост-обрезание дерева** - упрощение уже построенного дерева до заданной глубины.

### +ы & -ы решающих деревьев

> [!success] Преимущества
> *  Простая интерпретируемость.
> *  Отсутствие необходимости масштабирования данных.
> * Поддержка категориальных признаков.

> [!failure] Недостатки
> * Склонность к переобучению.
> * Неустойчивость (малое изменение данных меняет структуру дерева).
