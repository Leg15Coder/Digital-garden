==Опр.== **Ансамбль** - это модель, объединяющая прогнозы нескольких более простых, базовых алгоритмов $b_1(x), \dots, b_N(x)$.

### Виды ансамблей

1. **Bagging (Bootstrap Aggregating):** Обучение независимых моделей на случайных подмножествах данных (bootstrap) и усреднение ответов.

	Особенностью bagging является возможность распараллелить обучение: модели $b_i(x)$ предполагаются независимыми, поэтому каждую можно обучать параллельно с другой.

	==Опр.== **Процедура bootstrap** - создание $M$ независимых выборок из одной исходной, такого же как она размером, путём применения модели выбора элементов с возвращением (то есть элементы могут повторяться).

	==Опр.== **RSM (Random Subspace Method)** - bootstrap по признакам, а не объектам.

	==Опр.== **Pasting** - bootstrap без возвращения.

2. **Stacking:** Модели первого уровня обучаются на данных, а их прогнозы становятся признаками для мета-модели второго уровня.

3. **Blending:** Упрощенный stacking, где мета-модель - это взвешенная сумма прогнозов базовых моделей.

4. **Boosting:** Последовательное построение моделей, где каждая следующая исправляет ошибки предыдущих.
