Алгоритм градиентного спуска работает итеративно. Пусть $w_k$ - веса, полученные на $k$-й итерации алгоритма. Пусть $\eta : \mathbb{N}_0 \to \mathbb{R}$ - функция, определяющая темп обучения, а $\eta(k) = \eta_k$, тогда правило обновления весов* в полном градиентном спуске будет иметь вид:$$w_{k+1} = w_k - \eta_k \nabla \tilde{Q}(w_k)$$Отрицательный градиент в точке указывает на направление наиболее сильного уменьшения функции, то есть со временем веса сойдутся в точку локального минимума. Градиент функционала для [[Линейные модели|линейной модели]] выглядит так:$$\nabla \tilde{Q}(w) = \frac{1}{\ell}\sum_{i=1}^{\ell} y_i x_i \mathcal{L}'(M_i)$$

### Стохастический градиентный спуск (SGD)

При большом объёме выборки ($\ell >> 1$) возникает проблема с тем, что алгоритм слишком долго работает. Для ускорения обучения на больших данных используется стохастическая аппроксимация. Пусть $x_k$ - случайный вектор из $x$, выбранный на $k$-й итерации, а $y_k$ - соответствующее ему целевое значение, тогда$$w_{k+1} = w_k - \eta_k \nabla \mathcal{L}(w_k)$$В линейной модели:$$\nabla \mathcal{L}(w_k) = y_k x_k \mathcal{L}'(M_k)$$
### Теорема о статистической эквивалентности GD и SGD

Градиент, вычисленный методом SGD, является несмещённой оценкой истинного градиента GD: $$\mathbb{E}_{i}[\nabla \mathcal{L}_i(w)] = \nabla\tilde Q(w)$$
#### Доказательство

Рассмотрим математическое ожидание стохастического градиента по равномерно распределенному индексу $i$:$$\mathbb{E}_{i}(\nabla \mathscr{L}_i(w)) = \sum_{i=1}^{\ell} P(i) \cdot \nabla \mathscr{L}_i(w)$$При равномерном распределении $P(i) = \frac{1}{\ell}$, следовательно:$$\mathbb{E}_{i}(\nabla \mathscr{L}_i(w)) = \frac{1}{\ell} \sum_{i=1}^{\ell} \nabla \mathscr{L}_i(w) = \nabla Q(w)$$
#### Следствие

Правило обновления SGD можно представить как:$$w_{k+1} = w_k - \eta_k \nabla Q(w_k) + \eta_k \underbrace{(\nabla Q(w_k) - \nabla \mathscr{L}_{i_k}(w_k))}_{\varepsilon_k}$$где $\varepsilon_k$ - шум с нулевым математическим ожиданием: $\mathbb{E}\varepsilon_k = 0$.

### Теорема о скорости сходимости GD

Обозначим $w^*$ - оптимальный вектор весов. Пусть:
1) $Q$ - $\mu$-сильно выпуклая функция *(то есть $\forall w, w'\ :$ $Q(w') \ge Q(w) + \nabla Q(w)^T(w'-w) + \frac{\mu}{2}\|w'-w\|^2$)*
2) $\nabla Q$ - липшицева с константой Липшица $L$
3) $\forall k\ :\ \eta_k \le L^{-1}$
Тогда алгоритм GD обеспечивает линейную скорость сходимости:$$Q(w_k) - Q(w^*) \le \left(1 - \frac{\mu}{L}\right)^k (Q(w_0) - Q(w^*))$$
### Теорема о скорости сходимости SGD

Обозначим $w^*$ - оптимальный вектор весов. Пусть:
1) $Q$ - $\mu$-сильно выпуклая функция *(то есть $\forall w, w'\ :$ $Q(w') \ge Q(w) + \nabla Q(w)^T(w'-w) + \frac{\mu}{2}\|w'-w\|^2$)*
2) $\exists \sigma > 0\ :\ \mathbb{D}(\|\nabla Q\|) < \sigma^2$ - ограниченная дисперсия нормы градиента
3) $\forall k\ :\ \eta_k \le (\mu k)^{-1}$
Тогда алгоритм GD обеспечивает сублинейную скорость сходимости:$$\mathbb{E}(Q(w_k) - Q(w^*)) \le O\left(\frac{\sigma^2}{\mu k}\right), \quad k \to \infty$$
### Сравнительный анализ GD и SGD

|              Характеристика               |                         GD                         |                      SGD                       |
| :---------------------------------------: | :------------------------------------------------: | :--------------------------------------------: |
|         **Асимптотика итерации**          |                  $\Theta(\ell d)$                  |                  $\Theta(d)$                   |
|            **Память итерации**            |                  $\Theta(\ell d)$                  |                  $\Theta(d)$                   |
|          **Скорость сходимости**          |                    $O(e^{-k})$                     |                  $O(k^{-1})$                   |
| **Зависимость от точности $\varepsilon$** | $O\left(\log\frac{1}{\varepsilon}\right)$ итераций | $O\left(\frac{1}{\varepsilon}\right)$ итераций |
|              **Траектория**               |             Детерминированная, гладкая             |             Стохастическая, шумная             |
|          **Работа с выбросами**           |                      Устойчив                      |              Может раскачиваться               |
