# Вариации определения
## Двуаргументная ВМТ

- [[Машина Тьюринга|МТ]] $M(x, r)$ принимает на вход два аргумента на разных лентах, $x$ - на стандартной ленте, а $r$ - на специальной ленте, доступной только для чтения.

## ВМТ с вероятностной лентой

- [[Машина Тьюринга|МТ]] $M(x)$ имеет дополнительную ленту доступную только на последовательное чтение вправо, которую называют *вероятностной лентой*, заполненную случайными битами. Обычно значения бит на ленте распределены равномерно и независимо, то есть каждая по отдельности ячейка является результатом [[Схема Бернулли|схемы Бернулли]] с параметром $0.5$.

## ВМТ с двумя функциями перехода

-  [[Машина Тьюринга|МТ]] $M(x)$ имеет две функции перехода, и каждый такт работы она выбирает одну случайную из них.

## ВМТ с особым состоянием

- [[Машина Тьюринга|МТ]] $M(x)$ имеет одно или несколько специальных состояний, которые называют *вероятностные состояния*, после обработки вероятностного состояния под головкой МТ появляется случайный символ.

# Особенности ВМТ

- Результат работы ВМТ - [[Распределение случайных величин|вероятностное распределение]], индуцированное конкретным выбором входного слова и распределением значений $r$ (случайных бит)

- Время работы ВМТ - $\sup\limits_{r \in B^*}$ времени работы ДМТ $M(x, r)$

# Полиномиальная ВМТ

==Опр.== **ПВМТ** - ВМТ, которая работает за полиномиальное время.
