МНК позволяет найти оптимальные оценки параметра и ошибки в модели [[Линейная регрессия|линейной регрессии]]:$$\hat\theta = (Z^TZ)^{-1}Z^TX$$$$\hat{\sigma^2} = \frac{1}{n - k}\|X - Z\hat\theta\|_2^2$$Если модель регрессии гауссова, то распределения оценок будут иметь вид:$$\hat\theta \sim \mathcal{N}(\theta,\ (Z^TZ)^{-1}\sigma^2)$$$$\frac{\hat{\sigma^2}(n-k)}{\sigma^2} \sim \chi^2_{n-k}$$

$Q(\theta) = \|\varepsilon\|_2^2 = \|X - Z\theta\|_2^2 = (X - Z\theta)^T(X - Z\theta) \to \min\limits_{\theta}$
$Q(\theta) = X^TX - 2\theta^TZ^TX + \theta^TZ^TZ\theta$
$\frac{\partial Q(\theta)}{\partial \theta} = -2Z^TX + 2Z^TZ\theta = 0 \implies Z^TZ\theta = Z^TX$
В силу условия $\text{rk}(Z) = k$ матрица $(Z^TZ)$ обратима. Оценка МНК имеет вид:
$\hat{\theta} = (Z^TZ)^{-1}Z^TX$

Подставим в выражение для оценки уравнение модели $X = Z\theta + \varepsilon$:
$\hat{\theta} = (Z^TZ)^{-1}Z^T(Z\theta + \varepsilon) = (Z^TZ)^{-1}Z^TZ\theta + (Z^TZ)^{-1}Z^T\varepsilon = \theta + (Z^TZ)^{-1}Z^T\varepsilon$
Найдём математическое ожидание $\hat{\theta}$, учитывая, что ошибки центрированы ($\mathbb{E}\varepsilon = 0$):
$\mathbb{E}\hat{\theta} = \mathbb{E}\left[\theta + (Z^TZ)^{-1}Z^T\varepsilon\right] = \theta + (Z^TZ)^{-1}Z^T\mathbb{E}\varepsilon = \theta + 0 = \theta$
Следовательно, оценка $\hat{\theta}$ является несмещённой.

Найдём ковариационную матрицу $\mathbb{D}\hat{\theta}$, используя свойство $\mathbb{D}(AY) = A(\mathbb{D}Y)A^T$ и условие гомоскедастичности ошибок ($\mathbb{D}\varepsilon = \sigma^2 I_n$):
$\mathbb{D}\hat{\theta} = \mathbb{D}\left[\theta + (Z^TZ)^{-1}Z^T\varepsilon\right] = \mathbb{D}\left[(Z^TZ)^{-1}Z^T\varepsilon\right]$
$\mathbb{D}\hat{\theta} = (Z^TZ)^{-1}Z^T (\mathbb{D}\varepsilon) \left((Z^TZ)^{-1}Z^T\right)^T$
$\mathbb{D}\hat{\theta} = (Z^TZ)^{-1}Z^T (\sigma^2 I_n) Z (Z^TZ)^{-1} = \sigma^2 (Z^TZ)^{-1}Z^TZ(Z^TZ)^{-1} = \sigma^2(Z^TZ)^{-1}$

В гауссовской модели случайный вектор ошибок имеет многомерное нормальное распределение: $\varepsilon \sim \mathcal{N}(0, \sigma^2 I_n)$. Поскольку оценка представляет собой строго линейное преобразование вектора ошибок: $\hat{\theta} = \theta + (Z^TZ)^{-1}Z^T\varepsilon$ а любое линейное преобразование нормального вектора также является нормальным вектором, случайный вектор $\hat{\theta}$ подчиняется многомерному нормальному закону. Его параметрами являются ранее вычисленные математическое ожидание и ковариационная матрица: $\hat{\theta} \sim \mathcal{N}\left(\theta, (Z^TZ)^{-1}\sigma^2\right)$

Введём вектор остатков $e = X - Z\hat{\theta}$. Подставим в него выражение для ОМНК:
$e = X - Z(Z^TZ)^{-1}Z^TX = \left(I_n - Z(Z^TZ)^{-1}Z^T\right)X = (I_n - P)X$
где $P = Z(Z^TZ)^{-1}Z^T$ — матрица ортогонального проектора на подпространство столбцов матрицы $Z$.

Матрица $P$ обладает свойствами симметричности ($P^T = P$) и идемпотентности ($P^2 = P$). Отсюда матрица $I_n - P$ также симметрична и идемпотентна:
$(I_n - P)^T = I_n - P, \quad (I_n - P)^2 = I_n - P$
Выразим вектор остатков $e$ напрямую через случайные ошибки $\varepsilon$:
$e = (I_n - P)X = (I_n - P)(Z\theta + \varepsilon) = (Z\theta - PZ\theta) + (I_n - P)\varepsilon$
Так как $PZ = Z(Z^TZ)^{-1}Z^TZ = Z$, выражение в первых скобках тождественно равно нулю:
$e = (I_n - P)\varepsilon$

Тогда квадрат нормы остатков равен:
$\|X - Z\hat\theta\|_2^2 = \|e\|_2^2 = e^Te = \varepsilon^T(I_n - P)^T(I_n - P)\varepsilon = \varepsilon^T(I_n - P)\varepsilon$
Вычислим математическое ожидание квадрата нормы, используя свойства следа матрицы ($\text{tr}$):
$\mathbb{E}\|e\|_2^2 = \mathbb{E}\left[\text{tr}\left(\varepsilon^T(I_n - P)\varepsilon\right)\right] = \mathbb{E}\left[\text{tr}\left((I_n - P)\varepsilon\varepsilon^T\right)\right] = \text{tr}\left((I_n - P)\mathbb{E}[\varepsilon\varepsilon^T]\right)$
$\mathbb{E}\|e\|_2^2 = \text{tr}\left((I_n - P)\sigma^2 I_n\right) = \sigma^2 \text{tr}(I_n - P)$

След матрицы проектора равен рангу подпространства, на которое он проектирует. Поскольку $\text{tr}(I_n) = n$, а $\text{tr}(P) = \text{tr}(Z(Z^TZ)^{-1}Z^T) = \text{tr}((Z^TZ)^{-1}Z^TZ) = \text{tr}(I_k) = k$, получаем:
$\text{tr}(I_n - P) = n - k$
$\mathbb{E}\|e\|_2^2 = \sigma^2(n - k)$
Отсюда напрямую следует несмещённость оценки $\hat{\sigma^2}$:
$\mathbb{E}\hat{\sigma^2} = \frac{1}{n-k}\mathbb{E}\|e\|_2^2 = \frac{1}{n-k}\sigma^2(n-k) = \sigma^2$

Квадратичная форма имеет вид:
$\frac{\hat{\sigma^2}(n-k)}{\sigma^2} = \frac{\|e\|_2^2}{\sigma^2} = \left(\frac{\varepsilon}{\sigma}\right)^T (I_n - P) \left(\frac{\varepsilon}{\sigma}\right)$
Вектор $V = \frac{\varepsilon}{\sigma}$ является стандартным нормальным вектором: $V \sim \mathcal{N}(0, I_n)$.
Матрица $I_n - P$ является ортогональным проектором на подпространство ранга $n-k$. Из линейной алгебры известно, что любая матрица ортогонального проектора ранга $n-k$ имеет ровно $n-k$ собственных значений, равных $1$, и $k$ собственных значений, равных $0$.
По теореме о спектральном разложении, существует ортогональная матрица переходов $O$ ($O^TO = I_n$), которая диагонализирует проектор:
$I_n - P = O^T \Lambda O$
где $\Lambda = \text{diag}(1, 1, \dots, 1, 0, \dots, 0)$ — диагональная матрица, на главной диагонали которой расположено ровно $n-k$ единиц.
Подставим разложение в нашу квадратичную форму:
$\frac{\|e\|_2^2}{\sigma^2} = V^T O^T \Lambda O V = (OV)^T \Lambda (OV)$
Обозначим новый случайный вектор $\eta = OV$. Поскольку $V$ распределен нормально $\mathcal{N}(0, I_n)$, а $O$ — ортогональная матрица, это преобразование является поворотом пространства, сохраняющим независимость и дисперсию компонент:
$\mathbb{E}\eta = 0, \quad \mathbb{D}\eta = O (\mathbb{D}V) O^T = O I_n O^T = O O^T = I_n \implies \eta \sim \mathcal{N}(0, I_n)$

Теперь распишем квадратичную форму через компоненты вектора $\eta$:
$\frac{\|e\|_2^2}{\sigma^2} = \eta^T \Lambda \eta = \sum_{i=1}^n \lambda_i \eta_i^2 = \sum_{i=1}^{n-k} 1 \cdot \eta_i^2 + \sum_{j=n-k+1}^n 0 \cdot \eta_j^2 = \sum_{i=1}^{n-k} \eta_i^2$

Мы получили сумму квадратов $n-k$ независимых случайных величин, каждая из которых имеет стандартное нормальное распределение $\mathcal{N}(0, 1)$. По конструктивному определению распределения хи-квадрат, эта сумма строго подчиняется закону $\chi^2_{n-k}$: $\frac{\hat{\sigma^2}(n-k)}{\sigma^2} \sim \chi^2_{n-k}$

Оптимальность МНК-оценки $\hat{\theta}$ означает, что она обладает наименьшей дисперсией среди всех возможных линейных несмещённых оценок (свойство BLUE). Пусть $\tilde{\theta} = AX$ — любая другая линейная оценка параметра $\theta$, где $A$ — произвольная матрица весов размера $k \times n$.

1. Наложим условие несмещённости на оценку $\tilde{\theta}$: $\mathbb{E}\tilde{\theta} = \mathbb{E}[AX] = \mathbb{E}[A(Z\theta + \varepsilon)] = AZ\theta + A\mathbb{E}\varepsilon = AZ\theta$
    Чтобы равенство $\mathbb{E}\tilde{\theta} = \theta$ выполнялось при любых значениях вектора $\theta$, необходимо и достаточно, чтобы:
    $AZ = I_k$
    
2. Вычислим ковариационную матрицу оценки $\tilde{\theta}$: $\mathbb{D}\tilde{\theta} = \mathbb{D}(AX) = A(\mathbb{D}\varepsilon)A^T = \sigma^2 AA^T$
    
3. Представим матрицу $A$ в виде суммы МНК-оператора и некоторой матрицы отклонения $D$:
    $A = (Z^TZ)^{-1}Z^T + D$ где $D = A - (Z^TZ)^{-1}Z^T$.

Подставим это представление в условие несмещённости $AZ = I_k$:
$\left((Z^TZ)^{-1}Z^T + D\right)Z = I_k \implies (Z^TZ)^{-1}Z^TZ + DZ = I_k \implies I_k + DZ = I_k \implies DZ = 0$

4. Теперь раскроем матричное произведение $AA^T$:
    $AA^T = \left((Z^TZ)^{-1}Z^T + D\right)\left((Z^TZ)^{-1}Z^T + D\right)^T = \left((Z^TZ)^{-1}Z^T + D\right)\left(Z(Z^TZ)^{-1} + D^T\right)$
    $AA^T = (Z^TZ)^{-1}Z^TZ(Z^TZ)^{-1} + (Z^TZ)^{-1}Z^TD^T + DZ(Z^TZ)^{-1} + DD^T$

Применим полученное ранее условие $DZ = 0$ (откуда также следует $Z^TD^T = 0$). Перекрёстные сомножители зануляются: $AA^T = (Z^TZ)^{-1} + DD^T$

5. Умножим выражение на $\sigma^2$ для получения ковариационной матрицы:
    $\mathbb{D}\tilde{\theta} = \sigma^2(Z^TZ)^{-1} + \sigma^2 DD^T = \mathbb{D}\hat{\theta} + \sigma^2 DD^T$
    

Матрица $DD^T$ является неотрицательно определенной ($c^T DD^T c = \|D^T c\|^2 \ge 0$ для любого вектора $c$). Это означает, что для любой линейной комбинации параметров дисперсия альтернативной оценки $\tilde{\theta}$ будет больше или равна дисперсии МНК-оценки $\hat{\theta}$:
$\mathbb{D}(c^T\tilde{\theta}) = \mathbb{D}(c^T\hat{\theta}) + \sigma^2 \|D^T c\|^2 \ge \mathbb{D}(c^T\hat{\theta})$
Дисперсия достигает своего теоретического минимума тогда и только тогда, когда $D = 0$, что превращает $\tilde{\theta}$ в МНК-оценку $\hat{\theta}$. Оптимальность доказана.
