==Опр.== **Фазовый поток** - отображение $g^t: G \to G$, где $G$ - [[Фазовое пространство|фазовое пространство]], которое ставит в соответствие начальной точке $x_0$ значение решения $x(t)$ ЗК $\dot{x} = f(x), x(0) = x_0$ в момент времени $t$, то есть $g^t(x_0) = x(t)$

### Свойства фазового потока

1. $g^0 = id_G$ *(тождественное отображение)*
2. $g^{t+s} = g^t \circ g^s = g^s \circ g^t$
3. $(g^t)^{-1} = g^{-t}$
- Группа $\{g^t\}_{t \in \mathbb{R}}$ называется **однопараметрической группой диффеоморфизмов фазовых потоков** (ОГДФП)

- Доказательство свойств тривиально - путём прямой подстановки в определение

#### ОГДФП ОСЛУ

Для линейной системы $\dot{x} = Ax$ фазовый поток задается [[Матричная экспонента|матричной экспонентой]]: $g^t = e^{At}$
