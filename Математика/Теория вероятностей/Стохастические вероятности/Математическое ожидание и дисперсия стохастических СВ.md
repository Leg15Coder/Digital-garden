# Матожидание

==Опр.== **Математическое ожидание (матожидание) [[Стохастические случайные величины|СВ]] $\xi$** - $\mathbb{E}\xi = \int\limits_{\mathbb{R}}\xi dP$ $= \int\limits_{-\infty}^{+\infty}\xi(\omega) \cdot P(d\omega)$, причём данный интеграл *сходится абсолютно*. (Если интеграл не сходится или сходится условно, то матожидания не существует)

==Опр.== **$m$-й (абсолютный) момент** СВ $X$ - $\mathbb{E}X^m$

==Опр.== **$m$-й центральный момент** СВ $X$ - $\mathbb{E}(X^c)^m$, где $X^c$ - центровка $X$

### Свойства матожидания

1) Матожидание - функционал.
2) Матожидание - первый абсолютный момент.
3) Матожидание линейно. 
4) $\forall c \in \mathbb{R}\ :\ \mathbb{E}c = c$
5) $\forall A \in \mathcal{F}\ :\ \mathbb{E1}_A = P(A)$
6) $\forall \xi \geq 0\ :\ E\xi \geq 0$
7) $\xi \geq \eta\ \Longrightarrow\ \mathbb{E}\xi \geq \mathbb{E}\eta$ - монотонность
8) $\mathbb{E}(\xi\eta) \leq \sqrt{\mathbb{E}\xi^2 \cdot \mathbb{E}\eta^2}$ - неравенство КБШ. (Причём $\mathbb{E}(\xi\eta) = \sqrt{\mathbb{E}\xi^2 \cdot \mathbb{E}\eta^2}$ $\Leftrightarrow$ $\exists c = const,\ \eta = c \cdot \xi$)

# Дисперсия

==Опр.== **Дисперсия ДСВ $\xi$** - функционал вида $\mathbb{D}\xi = \mathbb{E}(\xi - \mathbb{E}\xi)^2 = \mathbb{E}\xi^2 - (\mathbb{E}\xi)^2$

==Опр.== $\sigma(\xi) = \sqrt{\mathbb{D}\xi}$

### Свойства дисперсии

1) Дисперсия - второй центральный момент.
2) $\mathbb{D}\xi = 0 \Leftrightarrow \xi = const$
3) $\forall c = const\ :\ \mathbb{D}(c\xi) = c^2 \mathbb{D}\xi$
4) Дисперсия НЕ линейна. (Контрпример: $\eta = 2\xi$)

- Центровка и нормировка СВ определяются так же, как в [[Математика/Теория вероятностей/Дискретное вероятностное пространство/Математическое ожидание и дисперсия|дискретном случае]]
