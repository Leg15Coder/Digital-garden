Пусть $\{\xi_i\}_{i=1}^n$ - НОРСВ $\sim Ber(p)$ (они могут иметь и другое распределение с конечной дисперсией, но теорему докажем только для бернуллевских СВ). Пусть а $S_n = \sum\limits_{k=1}^n \xi_k$, тогда$$\forall a,b \in \mathbb{R}\ :\ \lim\limits_{n \to \infty} P(a \leq S_n \leq b) = \frac{1}{\sqrt{2\pi}}\int\limits_{x_n}^{y_n}e^{-\frac{u^2}{2}}du = \Phi_0(y_n) - \Phi_0(x_n)$$где $x_n = \frac{a - np}{\sqrt{npq}}$, а $y_n = \frac{b - np}{\sqrt{npq}}$

#### Доказательство

Рассмотрим нормированную сумму:$$(S_n)^{cn} = \frac{S_n - \mathbb{E}S_n}{\sqrt{\mathbb{D}S_n}} = \frac{S_n - np}{\sqrt{npq}}$$
Применим [[Центральная предельная теорема|ЦПТ]]. Все условия выполнены:
1) $\{\xi_i\}$ НОРСВ по условию
2) $\mathbb{E}\xi_i = p$ и $\mathbb{D}\xi_i = pq$ существуют и конечны
Поэтому:$$(S_n)^{cn} \xrightarrow[n\to\infty]{d} \mathcal{N}(0,1)$$то есть$$\lim_{n\to\infty} P((S_n)^{cn} \leq x) = \Phi_0(x) = \frac{1}{\sqrt{2\pi}} \int_{-\infty}^x e^{-u^2/2} du$$
Преобразуем условие:$$P(a \leq S_n \leq b) = P\left(\frac{a - np}{\sqrt{npq}} \leq \frac{S_n - np}{\sqrt{npq}} \leq \frac{b - np}{\sqrt{npq}}\right) = P(x_n \leq (S_n)^{cn} \leq y_n)$$В силу равномерной сходимости по распределению:
$$\lim_{n\to\infty} P(x_n \leq (S_n)^{cn} \leq y_n) = \Phi_0(y_n) - \Phi_0(x_n)$$
Так как $x_n$ и $y_n$ зависят от $n$, то мы не можем напрямую применить определение сходимости для фиксированных границ. Однако заметим, что$$\Phi_0(y_n) - \Phi_0(x_n) = \frac{1}{\sqrt{2\pi}} \int_{x_n}^{y_n} e^{-u^2/2} du$$Таким образом мы получили то что нужно.

---
### Следствия

1. **Локальная теорема Муавра-Лапласа**:   $$P(S_n = k) \sim \frac{1}{\sqrt{2\pi npq}} e^{-\frac{(k-np)^2}{2npq}}, \quad n \to \infty$$
2. **Практическое применение.** При достаточно больших $n$ можно использовать приближение:$$P(a \leq S_n \leq b) \approx \Phi\left(\frac{b - np}{\sqrt{npq}}\right) - \Phi\left(\frac{a - np}{\sqrt{npq}}\right)$$
3. **Выражение через интеграл Лапласа:** $$\Phi_0(y) - \Phi_0(x) = \Phi\left(\frac{y - \mu}{\sigma}\right) - \Phi\left(\frac{x - \mu}{\sigma}\right)$$ где для для бернуллевских величин $\mu = np$, $\sigma = \sqrt{npq}$.


### Нормальная аппроксимация

> [!tip] Цель аппроксимации
> Мы хотим оценить скорость сходимости некоторого распределения к нормальному. Для этого строим оценки погрешностей приближения. Пусть $S_n = \sum\limits_{k=1}^n\xi_k$ и $F_{(S_n)^{cn}}(x) = F_n(x)$.

==Опр.== $\Delta_n = \sup\limits_{x \in \mathbb{R}}|F_n(x) - \Phi(x)|$

==Опр.== **Нормальная аппроксимация** - это замена исходного распределения случайной величины нормальным распределением с теми же математическим ожиданием и дисперсией.

> [!success] Способ аппроксимации
> Точность замены оценивается с помощью интегральной теоремы Муавра-Лапласа.
