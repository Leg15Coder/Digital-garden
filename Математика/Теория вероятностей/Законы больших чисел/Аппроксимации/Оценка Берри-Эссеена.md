Пусть $\{\xi_k\}_{k=1}^\infty$ - НОРСВ т.ч.:
1. $\mathbb{E}\xi_k = 0$
2. $\mathbb{D}\xi_k = \sigma^2 > 0$
3. $\mathbb{E}|\xi_k|^3 = \rho < \infty$ *(существует конечный абсолютный третий момент)*
Обозначим нормированную сумму:
$$S_n = \frac{1}{\sigma\sqrt{n}}\sum_{k=1}^n \xi_k$$

Пусть $F_n(x) = P(S_n \le x)$ - функция распределения $S_n$, тогда $\exists C > 0$ т.ч. для $\forall n \in \mathbb{N}$ и $\forall x \in \mathbb{R}$:$$\sup_{x \in \mathbb{R}} |F_n(x) - \Phi(x)| \le \frac{C\rho}{\sigma^3\sqrt{n}}$$
### Смысл

1. **Скорость сходимости**: Оценка показывает, что отклонение от [[Интегральная теорема Муавра-Лапласа|нормальности]] убывает как $O(\frac{1}{\sqrt{n}})$
2. **Равномерность по $x$**: Супремум берётся по всем $x$, то есть оценка работает для максимального отклонения графиков функций распределения

### Константа Берри-Эссеена

Константа $C$ в оценке - предмет многолетних исследований. Её значение улучшалось с годами:

| Год  | Значение   |
| ---- | ---------- |
| 1942 | 7.59       |
| 1972 | 0.7975     |
| 1986 | 0.7655     |
| 2007 | 0.7056     |
| 2009 | 0.5894     |
| 2010 | 0.5129     |
| 2011 | **0.4748** |

### Замечание о точности

Оценка $\sqrt{n}$ является **неулучшаемой** по порядку: существуют распределения (например, бернуллиевские), для которых$$\limsup_{n\to\infty} \sqrt{n} \sup_x |F_n(x) - \Phi(x)| > 0$$
то есть скорость $O(\frac{1}{\sqrt{n}})$ оптимальна. **Нижняя граница**:
$$C \ge \frac{\sqrt{10} + 3}{6\sqrt{2\pi}} \approx 0.40973 = C_0$$

### Обобщение

Пусть $\{\xi_k\}_{k=1}^\infty$ - независимые величины с $\mathbb{E}\xi_k = 0$, $\mathbb{E}\xi_k^2 = \sigma_k^2 > 0$, $\mathbb{E}|\xi_k|^3 = \rho_k < \infty$. Обозначим:$$s_n^2 = \sum_{k=1}^n \sigma_k^2, \quad r_n = \sum_{k=1}^n \rho_k$$
Тогда $\exists$ абсолютная константа $C_0$ т.ч.:
$$\sup_{x \in \mathbb{R}} \left| P\left(\frac{\sum_{k=1}^n \xi_k}{s_n} \le x\right) - \Phi(x) \right| \le C_0 \frac{r_n}{s_n^3}$$
При этом $C_0$ удовлетворяет той же нижней границе $C_0 \ge 0.4097$
