### Теорема о связи замкнутости и полноты

Пусть $\{e_k\}_{k=1}^\infty$ - ОНС в гильбертовом пространстве $H$. Система является полной \iff она замкнута.

#### Доказательство Т. 4

1. Необходимость ($\Rightarrow$). Пусть система полна. Возьмём произвольный вектор $x \in H$, ортогональный всей системе, т.ч. $\forall k \in \mathbb{N} : (x, e_k) = 0$.
    
2. Поскольку система полна, для неё справедливо равенство Парсеваля (Т. 3) : $\|x\|^2 = \sum\limits_{k=1}^\infty |(x, e_k)|^2$.
    
3. Подставляя нулевые значения скалярных произведений, получаем : $\|x\|^2 = 0 \Rightarrow x = 0$. Условие замкнутости доказано.
    
4. Достаточность ($\Leftarrow$). Пусть система замкнута. $\forall x \in H$ составим последовательность его коэффициентов Фурье $c_k = (x, e_k)$.
    
5. По Т. Риса-Фишера из сходимости ряда квадратов $\sum\limits_{k=1}^\infty |c_k|^2 < \infty$ вытекает существование предела ряда Фурье по норме пространства : $\exists \tilde{x} \in H : \lim\limits_{n\to\infty} \sum\limits_{k=1}^n c_k e_k = \tilde{x}$.
    
6. Согласно Т. о перпендикулярности, вектор разности $h = x - \tilde{x}$ обладает свойством ортогональности ко всем элементам ОНС : $\forall k \in \mathbb{N} : (h, e_k) = 0$.
    
7. Из исходного предположения о замкнутости системы : $h = 0 \Rightarrow x = \tilde{x}$.
    
8. Таким образом, предел частичных сумм совпадает с $x$, т.ч. $x \in \overline{\text{span}(\{e_k\}_{k=1}^\infty)}$. Система полна.
