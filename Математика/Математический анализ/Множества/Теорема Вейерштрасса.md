### 1-я теорема Вейерштрасса

Пусть $f \in C_{[-\pi; \pi]}$ т.ч. $f(-\pi) = f(\pi)$, тогда $\forall \varepsilon > 0\ \exists T_n \in \text{span}(Trig) =$ $\frac{a_0}{2} + \sum\limits_{k=1}^n (a_k \cos kx + b_k \sin kx)\ :\ \|f - T_n\|_c =$ $\sup\limits_{x \in [-\pi;\pi]}|f(x) - T_n(x)| < \varepsilon$

#### Доказательство

1. Пусть $f \in C_{2\pi}(\mathbb{R})$. По Т. Фейера последовательность сумм Фейера $\sigma_n(x; f)$ сходится к $f(x)$ равномерно на всей вещественной оси $\mathbb{R}$.
    
2. Из определения равномерной сходимости : $\forall \epsilon > 0 \exists N \in \mathbb{N} : \forall n \ge N \Rightarrow \max\limits_{x \in \mathbb{R}} |f(x) - \sigma_n(x; f)| < \epsilon$.
    
3. По определению $\sigma_n(x; f)$ является линейной комбинацией частичных сумм Фурье $S_k(x; f)$, т.ч. она представляет собой тригонометрический полином порядка $n$.
    
4. Выбирая $T_n(x) = \sigma_N(x; f)$, получаем искомое неравенство : $\max\limits_{x \in \mathbb{R}} |f(x) - T_n(x)| < \epsilon$.
    

Т. 2 (Вторая Т. Вейерштрасса): $\forall f \in C[a, b]\ \forall \epsilon > 0 \exists P_n(x) : \max\limits_{x \in [a, b]} |f(x) - P_n(x)| < \epsilon$.

### 2-я теорема Вейерштрасса

$\forall f \in C_{[a; b]}\ \exists P_n \in \mathbb{P}_\infty[x]\ :\ \|f - P_n\|_c =$ $\sup\limits_{x \in [a;b]}|f(x) - P_n(x)| < \varepsilon$

#### Доказательство

1. С помощью линейной замены переменной переведём отрезок $[a, b]$ в $[0, \pi]$. Продолжим функцию $f \in C[0, \pi]$ чётным путём на отрезок $[-\pi, 0]$, а затем распространим её по периодичности на всю числовую прямую $\Rightarrow f \in C_{2\pi}(\mathbb{R})$.
    
2. По Т. 1 $\forall \epsilon > 0 \exists T_n(x) : \max\limits_{x \in \mathbb{R}} |f(x) - T_n(x)| < \frac{\epsilon}{2}$.
    
3. Каждый базисный элемент $\cos kx$ и $\sin kx$, входящий в состав тригонометрического полинома $T_n(x)$, разложим в ряд Маклорена, который сходится равномерно на любом компакте.
    
4. Заменяя тригонометрические функции частичными суммами их рядов Маклорена достаточно высокого порядка, сформируем алгебраический многочлен $P_m(x) : \max\limits_{x \in [0, \pi]} |T_n(x) - P_m(x)| < \frac{\epsilon}{2}$.
    
5. По неравенству треугольника : $\max\limits_{x \in [0, \pi]} |f(x) - P_m(x)| \le \max\limits_{x \in [0, \pi]} |f(x) - T_n(x)| + \max\limits_{x \in [0, \pi]} |T_n(x) - P_m(x)| < \epsilon$.
