### 1-я теорема Вейерштрасса

Пусть $f \in C_{[-\pi; \pi]}$ т.ч. $f(-\pi) = f(\pi)$, тогда $\forall \varepsilon > 0\ \exists T_n \in \text{span}(Trig) =$ $\frac{a_0}{2} + \sum\limits_{k=1}^n (a_k \cos kx + b_k \sin kx)\ :\ \|f - T_n\|_c =$ $\sup\limits_{x \in [-\pi;\pi]}|f(x) - T_n(x)| < \varepsilon$

#### Доказательство

По Т. Фейера последовательность сумм Фейера $\sigma_n(x; f)$ сходится к $f(x)$ равномерно на $\mathbb{R}$. Из определения равномерной сходимости : $\forall \varepsilon > 0\ \exists N \in \mathbb{N}\ \forall n \ge N\ :\ \sup\limits_{x \in \mathbb{R}} |f(x) - \sigma_n(x; f)| < \varepsilon$. По определению $\sigma_n(x; f)$ является линейной комбинацией частичных сумм Фурье $S_k(x; f)$, т.ч. она представляет собой тригонометрический полином порядка $n$. Выбирая $T_n(x) = \sigma_N(x; f)$, получаем искомое неравенство : $\sup\limits_{x \in \mathbb{R}} |f(x) - T_n(x)| < \varepsilon$.

### 2-я теорема Вейерштрасса

$\forall f \in C_{[a; b]}\ \exists P_m \in \mathbb{P}_\infty[x]\ :\ \|f - P_m\|_c =$ $\sup\limits_{t \in [a;b]}|f(t) - P_n(t)| < \varepsilon$

#### Доказательство

1. С помощью линейной замены переменной переведём отрезок $[a, b]$ в $[0, \pi]$. Продолжим функцию $f \in C[0, \pi]$ чётным путём на отрезок $[-\pi, 0]$, а затем распространим её по периодичности на всю числовую прямую. Формально:
	1. $\forall t \in [a; b]\ :\ f(x) := f(a + \frac{b - a}{\pi}t)$
	2. $\forall x \in [-\pi; 0)\ :\ f(x) := f(-x)$
	3. $\forall x \in \mathbb{R}\ :\ f(x) := f(x + 2\pi)$

2. По 1-й Т. В.-Ш. $\forall \varepsilon > 0\ \exists T_n(x)\ :\ \sup\limits_{x \in \mathbb{R}} |f(x) - T_n(x)| < \frac{\varepsilon}{2}$. Каждый базисный элемент $\cos kx$ и $\sin kx$, входящий в состав $T_n(x)$, разложим в ряд Маклорена, который для данных функций сходится равномерно на любом компакте. Это значит, что $\exists m \in \mathbb{N}\ \exists P_m(x) \in \mathbb{P}[x]\ :\ \sup\limits_{x \in [0, \pi]} |T_n(x) - P_m(x)| < \frac{\varepsilon}{2}$

3. По неравенству треугольника : $\sup\limits_{x \in [0, \pi]} |f(x) - P_m(x)| \le$ $\sup\limits_{x \in [0, \pi]} |f(x) - T_n(x)| + \sup\limits_{x \in [0, \pi]} |T_n(x) - P_m(x)|$ $< \varepsilon$.

4. Используя обратное преобразование $P_m(t) := P_m(\frac{x - a}{b - a}\pi)$, получаем искомый полином, который приближает исходную $f$ на отрезке $[a; b]$ с точностью $\varepsilon$.
