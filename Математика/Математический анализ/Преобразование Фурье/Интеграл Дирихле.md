В качестве примера применения [[Теорема Римана-Лебега об осцилляции|Т. Р-Л]] об осцилляции рассмотрим интеграл Дирихле:$$I = \int\limits_0^{+\infty} \frac{\sin x}{x}dx$$Поскольку обе границы интегрирования являются неопределённостями, то сложно посчитать данное значение в лоб. Распишем несобственный интеграл по определению:$$I = \lim\limits_{n \to+\infty}\int\limits_{0}^{(n + \frac{1}{2})\pi}\frac{\sin x}{x}dx$$Сделаем замену $x = (n + \frac{1}{2})t$:$$I = \lim\limits_{n \to+\infty}\int\limits_{0}^{\pi}\frac{\sin (n + \frac{1}{2})t}{t}dt = \lim\limits_{n \to+\infty} \left( \int\limits_{0}^{\pi}(\frac{1}{t} - \frac{1}{2\sin\frac{t}{2}})\cdot \sin (n + \frac{1}{2})t \cdot dt + \int\limits_{0}^{\pi}\frac{\sin (n + \frac{1}{2})t}{2\sin\frac{t}{2}} dt \right)$$Первый интеграл удовлетворяет Т. Р-Л об осцилляции, поскольку функция $f(t) = (\frac{1}{t} - \frac{1}{2\sin\frac{t}{2}}) \in \mathcal{L}^1(\mathbb{R})$. Посчитаем второй интеграл:$$I = \lim\limits_{n \to+\infty}\int\limits_{0}^{\pi}\frac{\sin (n + \frac{1}{2})t}{2\sin\frac{t}{2}} dt = \lim\limits_{n \to+\infty} \int\limits_{0}^{\pi} \pi D_n(t) dt = \frac{\pi}{2}\lim\limits_{n \to+\infty} \int\limits_{-\pi}^{\pi}D_n(t) dt = \frac{\pi}{2}$$Таким образом мы смогли взять несобственный интеграл.
