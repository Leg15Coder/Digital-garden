# 2.3) Поточечная сходимость ряда Фурье

## Принцип локализации

Пусть $f, g \in \mathcal{L}^1([-\pi; \pi])$.

Утв. Принцип локализации
Из предположения локального совпадения функций : $\exists \delta > 0 \ \forall x \in [x_0 - \delta; x_0 + \delta] \ : \ f(x) = g(x) \ \Rightarrow \ \lim\limits_{n \to \infty} (S_n[f](x_0) - S_n[g](x_0)) = 0$

#### Доказательство
1. Интегральное представление разности частичных сумм через ядро Дирихле:
$$S_n[f](x_0) - S_n[g](x_0) = \int\limits_{-\pi}^{\pi} (f(x_0 - t) - g(x_0 - t)) D_n(t) \, dt$$
2. Т.к. $\forall t \in [-\delta; \delta] \ : \ f(x_0 - t) - g(x_0 - t) = 0$, то интеграл по окрестности нуля зануляется:
$$S_n[f](x_0) - S_n[g](x_0) = \int\limits_{\delta \le |t| \le \pi} (f(x_0 - t) - g(x_0 - t)) D_n(t) \, dt$$
3. Подставим явный вид ядра Дирихле:
$$S_n[f](x_0) - S_n[g](x_0) = \int\limits_{\delta \le |t| \le \pi} \frac{f(x_0 - t) - g(x_0 - t)}{4\pi \sin\limits\left(\frac{t}{2}\right)} \sin\limits\left(\left(n + \frac{1}{2}\right)t\right) \, dt$$
4. Введём вспомогательную функцию:
$$F(t) = \begin{cases} \frac{f(x_0 - t) - g(x_0 - t)}{4\pi \sin\limits\left(\frac{t}{2}\right)}, & \delta \le |t| \le \pi \\ 0, & |t| < \delta \end{cases}$$
5. Знаменатель отделён от нуля, т.к. $\forall t \in [-\pi; \pi] \setminus [-\delta; \delta] \ : \ |\sin\limits\left(\frac{t}{2}\right)| \ge \sin\limits\left(\frac{\delta}{2}\right) > 0 \ \Rightarrow \ F \in \mathcal{L}^1([-\pi; \pi])$.
6. По Т. Римана-Лебега об осцилляции при $\omega = n + \frac{1}{2} \to \infty$ выполняется :
$$\lim\limits_{n \to \infty} \int\limits_{-\pi}^{\pi} F(t) \sin\limits\left(\left(n + \frac{1}{2}\right)t\right) \, dt = 0$$

## Явление Гиббса

==Опр.== **Явление Гиббса** - эффект незатухающих колебаний частичных сумм ряда Фурье в окрестности точки разрыва первого рода функции, при котором величина всплеска сохраняет ненулевой предел при росте частоты.

Рассмотрим модельный пример: $f(x)$ $=$ $\text{sgn}(x)$ на $[-\pi; \pi]$. Её формальный ряд Фурье имеет вид:
$$S[f](x) = \frac{4}{\pi} \sum\limits_{k=1}^\infty \frac{\sin((2k-1)x)}{2k-1}$$

Утв. Предельный наплыв сумм
Частичная сумма РФ для знаковой функции представима интегралом Дирихле, а её первый экстремум при $n$ $\to$ $\infty$ стремится к константе Гиббса.

#### Доказательство
1. Запишем выражение для частичной суммы $S_{2n}[f](x)$ через тригонометрическое тождество:
$$S_{2n}[f](x) = \frac{2}{\pi} \int\limits_0^x \frac{\sin(2nt)}{\sin t} \, dt$$
2. Найдём точку первого локального максимума через производную : \frac{d}{dx} S_{2n}[f](x) = \frac{2}{\pi} \frac{\sin(2nx)}{\sin x} = 0 \ \Rightarrow \ x_n = \frac{\pi}{2n}.
3. Подставим критическую точку $x_n$ в интеграл:
$$S_{2n}[f]\left(\frac{\pi}{2n}\right) = \frac{2}{\pi} \int\limits_0^{\frac{\pi}{2n}} \frac{\sin(2nt)}{\sin t} \, dt$$
4. Сделаем замену переменной $\tau$ $=$ $2nt$ \ : \ dt = \frac{d\tau}{2n}:
$$S_{2n}[f]\left(\frac{\pi}{2n}\right) = \frac{2}{\pi} \int\limits_0^\pi \frac{\sin \tau}{\sin\left(\frac{\tau}{2n}\right)} \frac{d\tau}{2n}$$
5. Перейдём к пределу при $n$ $\to$ $\infty$, используя эквивалентность $\sin\left(\frac{\tau}{2n}\right)$ $\sim$ $\frac{\tau}{2n}$:
$$\lim\limits_{n \to \infty} S_{2n}[f]\left(\frac{\pi}{2n}\right) = \frac{2}{\pi} \int\limits_0^\pi \frac{\sin \tau}{\tau} \, d\tau \approx 1.17897$$

> [!IMPORTANT]
> Величина полувсплеска над уровнем скачка составляет приближённо $18\%$ от величины полускачка функции (суммарный скачок осцилляции составляет около $118\%$ от исходного разрыва функции $d$).
