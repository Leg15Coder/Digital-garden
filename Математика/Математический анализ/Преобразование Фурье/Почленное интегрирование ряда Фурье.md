Пусть $f \in \mathcal{L}^1_{[-\pi; \pi]}$ - $2\pi$-периодичная функция с коэффициентами РФ $c_k[f]$, тогда $\forall x \in \mathbb{R}$ функция её первообразной $F(x) = \int\limits_0^x f(t) dt$ выражается функциональным рядом:$$F(x) = c_0[f] x + \sum\limits_{k=-\infty, k \neq 0}^{\infty} \frac{c_k[f]}{ik} \left( e^{ikx} - 1 \right)$$причём данный ряд сходится равномерно на всём $\mathbb{R}$, даже если исходный РФ функции $f$ всюду расходится.

#### Доказательство

1. Введём вспомогательную функцию $G(x) = \int\limits_0^x f(t) dt - c_0[f] x$. Проверим её периодичность на периоде $2\pi$:$$G(x + 2\pi) - G(x) = \int\limits_x^{x+2\pi} f(t) dt - 2\pi c_0[f] = \int\limits_{-\pi}^{\pi} f(t) dt - 2\pi \cdot \left( \frac{1}{2\pi} \int\limits_{-\pi}^{\pi} f(t) dt \right) = 0 \quad \Longrightarrow \quad G(x+2\pi) = G(x)$$
2. Т.к. $f \in \mathcal{L}^1_{[-\pi; \pi]}$, то $G \in C(\mathbb{R})$, а её производная $G'(x) = f(x) - c_0[f]$ кусочно-непрерывна на $[-\pi; \pi]$.
3. Из непрерывности и кусочной гладкости $G(x)$ следует, что её собственный РФ сходится к ней равномерно на $\mathbb{R}$:$$G(x) = \sum\limits_{k=-\infty}^{\infty} c_k[G] e^{ikx}$$
4. Свяжем коэффициенты $c_k[G]$ с коэффициентами производной $G'$ при $k \neq 0$ через [[Интегрирование ряда Фурье по частям|Т. о связи коэффициентов при дифференцировании]]:$$c_k[G] = \frac{c_k[G']}{ik}$$
5. Из выражения $G'(x) = f(x) - c_0[f]$ имеем, что $\forall k \neq 0 \ : \ c_k[G'] = c_k[f]$, откуда:$$\forall k \neq 0 \ : \ c_k[G] = \frac{c_k[f]}{ik}$$
6. Коэффициент $c_0[G]$ определим из начального условия $G(0) = 0$:$$G(0) = c_0[G] + \sum\limits_{k=-\infty, k \neq 0}^{\infty} \frac{c_k[f]}{ik} = 0 \quad \Longrightarrow \quad c_0[G] = - \sum\limits_{k=-\infty, k \neq 0}^{\infty} \frac{c_k[f]}{ik}$$
7. Подставим полученные выражения в равномерно сходящийся ряд для $G(x)$:$$G(x) = - \sum\limits_{k=-\infty, k \neq 0}^{\infty} \frac{c_k[f]}{ik} + \sum\limits_{k=-\infty, k \neq 0}^{\infty} \frac{c_k[f]}{ik} e^{ikx} = \sum\limits_{k=-\infty, k \neq 0}^{\infty} \frac{c_k[f]}{ik} \left( e^{ikx} - 1 \right)$$Из определения $F(x) = G(x) + c_0[f] x$ получаем искомое разложение с гарантированной равномерной сходимостью функционального ряда на $\mathbb{R}$.
