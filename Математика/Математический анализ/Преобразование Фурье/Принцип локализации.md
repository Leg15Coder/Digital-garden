Пусть $f, g \in \mathcal{L}^1_{[-\pi; \pi]}$ и пусть $x_0 \in (-\pi; \pi)$. Если $\exists \delta > 0 \ \forall x \in U_\delta(x_0)\ : \ f(x) = g(x)$, то $\lim\limits_{n \to \infty} (S_n[f](x_0)) = \lim\limits_{n \to \infty} (S_n[g](x_0))$ *(при условии, что данные пределы существуют)*

#### Доказательство

1. Интегральное представление разности частичных сумм через ядро Дирихле:$$S_n[f](x_0) - S_n[g](x_0) = \int\limits_{-\pi}^{\pi} (f(x_0 - t) - g(x_0 - t)) D_n(t) \, dt$$
2. Т.к. $\forall t \in [-\delta; \delta] \ : \ f(x_0 - t) - g(x_0 - t) = 0$, то интеграл по окрестности нуля зануляется:$$S_n[f](x_0) - S_n[g](x_0)\ \ \ = \int\limits_{\delta \le |t| \le \pi} (f(x_0 - t) - g(x_0 - t)) D_n(t) \, dt$$
3. Подставим явный вид ядра Дирихле:$$S_n[f](x_0) - S_n[g](x_0)\ \  \ = \int\limits_{\delta \le |t| \le \pi} \frac{f(x_0 - t) - g(x_0 - t)}{4\pi \sin\limits\left(\frac{t}{2}\right)} \sin\limits\left(\left(n + \frac{1}{2}\right)t\right) \, dt$$
4. Введём вспомогательную функцию:$$F(t) = \frac{f(x_0 - t) - g(x_0 - t)}{4\pi \sin\limits\left(\frac{t}{2}\right)} \mathbb{1}_{\delta \le |t| \le \pi}(t)$$
5. Знаменатель отделён от нуля, т.к. $\forall t \in [-\pi; \pi] \setminus [-\delta; \delta] \ : \ |\sin\limits\left(\frac{t}{2}\right)| \ge \sin\limits\left(\frac{\delta}{2}\right) > 0$ $\Rightarrow$ $F \in \mathcal{L}^1_{[-\pi; \pi]}$
6. По [[Теорема Римана-Лебега об осцилляции|Т. Р-Л об осцилляции]] при $\omega = n + \frac{1}{2} \to \infty$ выполняется :$$\lim\limits_{n \to \infty} (S_n[f](x_0) - S_n[g](x_0)) = \lim\limits_{n \to \infty} \int\limits_{-\pi}^{\pi} F(t) \sin\limits\left(\left(n + \frac{1}{2}\right)t\right) \, dt = 0$$
