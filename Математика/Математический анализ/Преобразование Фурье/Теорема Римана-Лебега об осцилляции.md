Пусть $f \in \mathcal{L}^1(\mathbb{R})$, тогда:$$\lim\limits_{\omega \to \infty} \int\limits_{\mathbb{R}} f(x) \cdot e^{i\omega x} \, dx = 0$$В рациональной записи это эквивалентно:$$\lim\limits_{\omega \to \infty} \int\limits_{\mathbb{R}} f(x) \sin(\omega x) \, dx = 0 \quad \text{и} \quad \lim\limits_{\omega \to \infty} \int\limits_{\mathbb{R}} f(x) \cos(\omega x) \, dx = 0$$
#### Доказательство

1. Докажем для непрерывно дифференцируемых функций с компактным носителем. То есть предположим, что $supp(f) \subseteq [-R; R]$ и $f \in C'_{[-R; R]}$. Применим интегрирование по частям:$$\left| \int\limits_{-R}^{R} f(x) e^{i\omega x} \, dx \right| = \left| -\frac{f(x) e^{i\omega x}}{i\omega} \bigg|_{-R}^{R} + \frac{1}{\omega} \int\limits_{-R}^{R} f'(x) e^{i\omega x} \, dx \right| \le \frac{1}{|\omega|}\left(|f(R)| + |f(-R)| + \int\limits_{-R}^{R} |f'(x)| dx \right)$$Функции $f(x)$ и $f'(x)$ непрерывны на компакте $[a, b]$ $\Rightarrow$ они равномерно ограничены некоторой константой $C$. Отсюда при условии, что $\omega$ $\to$ $\infty$ правая часть стремится к нулю, и по предельному переходу исходный интеграл тоже обнуляется.

2. Докажем для индикаторных функций. Пусть $I = (a; b) \subset \mathbb{R}$ - промежуток. Пусть $f = \mathbb{1}_I$, тогда:$$\int\limits_{\mathbb{R}} f(x) e^{i\omega x} \, dx = \frac{e^{i\omega b} - e^{i\omega a}}{i\omega} \quad \xrightarrow[\omega \to \infty]{\text{огр. на беск. малую}} \quad 0$$
3. Для простых функций теорема выполняется в силу аддитивности интеграла. Пусть $f = \sum\limits_{k=1}^n \alpha_k \mathbb{1}_{I_k}$, где $I_k$ - промежутки, тогда:$$\int\limits_{\mathbb{R}} f(x) e^{i\omega x} \, dx = \sum \limits_{k=1}^n\alpha_k\int\limits_{\mathbb{R}}\mathbb{1}_{I_k}(x)dx \quad \xrightarrow[\omega \to \infty]{\text{сумма беск. малых}} \quad 0$$
4. Пусть теперь $f \in \mathcal{L}^1(\mathbb{R})$ рассмотрим простую функцию $\varphi = \sum\limits_{k=1}^n \alpha_k \mathbb{1}_{I_k}$ т.ч. $\|f - \varphi\|_1 < \frac{\varepsilon}{2}$, тогда по нер-ву $\triangle$-ка:$$\left| \int\limits_{\mathbb{R}} f(x) e^{i\omega x} \, dx \right| \le \int\limits_{\mathbb{R}}| (f(x) - \varphi(x) + \varphi(x)) e^{i\omega x} | dx \le \int\limits_{\mathbb{R}}| (f(x) - \varphi(x)| dx + \int\limits_{\mathbb{R}}| \varphi(x)| dx = \|f - \varphi\|_1 + \frac{\varepsilon}{2} < \varepsilon$$А т.к. можно взять $\forall \varepsilon > 0$, то интеграл зануляется.

> [!IMPORTANT] Поведение коэффициентов Фурье
> Из Т. Римана-Лебега напрямую вытекает фундаментальное свойство [[Ряд Фурье|коэффициентов РФ]] любой абсолютно интегрируемой функции: **при возрастании частоты гармоник их амплитуда зануляется**.
