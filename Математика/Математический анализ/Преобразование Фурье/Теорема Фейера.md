==Опр.== **Суммы Фейера** _(СФ)_ - функциональная последовательность $\sigma_n[f](x)$, определённая как среднее арифметическое первых $n + 1$ частичных сумм РФ функции $f$:$$\sigma_n[f](x) = \frac{1}{n+1} \sum\limits_{k=0}^{n} S_k[f](x)$$
==Опр.== **Ядро Фейера** - функциональная тригонометрическая сумма $\Phi_n(t)$, представляющая собой среднее арифметическое ядер Дирихле $D_k(t)$ :$$\Phi_n(t) = \frac{1}{n + 1} \sum\limits_{k=0}^{n} D_k(t)$$Связь понятий:$$\sigma_n(x; f) = \frac{1}{2\pi} \int\limits_{-\pi}^{\pi} f(x - t) \Phi_n(t) dt$$Аналитическое выражение ядра в замкнутой форме *(прямая подстановка [[Ряд Фурье|ядра Дирихле]] в определение ядра Фейера)*:$$\Phi_n(t) = \begin{cases} \frac{1}{4\pi (n+1)} \cdot \frac{\sin^2 \frac{(n+1)t}{2}}{\sin^2 \frac{t}{2}}, & t \neq 2\pi m, \ m \in \mathbb{Z} \\\\ 2n+1, & t = 2\pi m, \ m \in \mathbb{Z}\end{cases}$$

### Свойства ядра Фейера

1. $\forall t \in \mathbb{R} \ \forall n \in \mathbb{N} \ : \ F_n(t) \ge 0$.
2. Информированность интеграла:$$\int\limits_{-\pi}^{\pi} |\Phi_n(t)| dt = 1$$
3. Сосредоточенность меры вокруг нуля: $$\forall \delta > 0\ \forall t \in [\delta; \pi] \ : \ \Phi_n(t) \le \frac{1}{n \sin^2 \frac{\delta}{2}} \xrightarrow{n \to \infty} 0$$
#### Доказательство

1) Очевидно из аналитического выражения *(все функции в котором неотрицательны)*.
2) Используя нормированность ядра Дирихле:$$\int\limits_{-\pi}^{\pi}|\Phi_n(t)|dt = \int\limits_{-\pi}^{\pi}\Phi_n(t)dt = \frac{1}{n + 1} \sum\limits_{k=0}^{n} \int\limits_{-\pi}^{\pi} D_k(t) = 1$$
3) Рассмотрим:$$\int_\delta^\pi \Phi_n(x)dx \le \int_\delta^\pi \frac{1}{2\pi(n+1)}\frac{1}{\sin^2\frac{\delta}{2}} = O((n+1)^{-1}), \quad n \to \infty$$
> [!TIP] Смысл ядра Фейера
> Ключевое отличие СФ от классических частичных сумм РФ заключается в неотрицательности ядра $F_n(t)$ $\ge$ $0$. Это свойство полностью предотвращает явление Гиббса на разрывах её графика.

### Теорема Фейера

Пусть $f$ $\in$ $C(\mathbb{R})$ - $2\pi$-периодичная функция, тогда её СФ равномерно сходятся к функции: $\sigma_n[f](x)$ $\rightrightarrows$ $f(x)$ на $\mathbb{R}$ при $n$ $\to$ $\infty$.

#### Доказательство

1. Представление разности через нормированность ядра Фейера:$$\sigma_n(x; f) - f(x) = \int\limits_{-\pi}^{\pi} \big( f(x - t) - f(x) \big) F_n(t) dt$$
2. По Т. Кантора о равномерной непрерывности функции $f$ на компакте:$$\forall \varepsilon > 0 \ \exists \delta > 0 \ \forall x \in \mathbb{R} \ \forall t \in [-\delta; \delta] \ : \ |f(x - t) - f(x)| < \frac{\varepsilon}{2}$$
3. Декомпозиция интеграла по областям $[-\pi; \pi]$ $=$ $[-\pi; -\delta] \cup [-\delta; \delta] \cup [\delta; \pi]$:$$\big| \sigma_n(x; f) - f(x) \big| \le \left( \int\limits_{-\pi}^{-\delta} + \int\limits_{\delta}^{\pi} \right) |f(x - t) - f(x)| F_n(t) dt + \int\limits_{-\delta}^{\delta} |f(x - t) - f(x)| F_n(t) dt$$
4. Оценка центрального интеграла за счёт малости приращения функции:$$\int\limits_{-\delta}^{\delta} |f(x - t) - f(x)| F_n(t) dt < \frac{\varepsilon}{2}\int\limits_{-\delta}^{\delta} F_n(t) dt \le \frac{\varepsilon}{2} \cdot 1 = \frac{\varepsilon}{2}$$
5. Оценка боковых интегралов через равномерную ограниченность непрерывной функции $|f(x)|$ $\le$ $M$ на компакте:$$\left( \int\limits_{-\pi}^{-\delta} + \int\limits_{\delta}^{\pi} \right) 2M \cdot F_n(t) dt \le 2M \cdot \left( \sup\limits_{t \in [\delta; \pi]} F_n(t) \right) \cdot \frac{2\pi}{2\pi} \le \frac{2M}{n \sin^2 \frac{\delta}{2}}$$
6. Зафиксируем пороговый индекс:$$\exists N \in \mathbb{N} \ \forall n \ge N\ :\ \frac{2M}{n \sin^2 \frac{\delta}{2}} < \frac{\varepsilon}{2}$$
7. Тогда:$$\forall n \ge N \ \forall x \in \mathbb{R} \ : \ |\sigma_n(x; f) - f(x)| < \frac{\varepsilon}{2} + \frac{\varepsilon}{2} = \varepsilon \quad \Longrightarrow \quad \sigma_n(x; f) \rightrightarrows f(x)$$
