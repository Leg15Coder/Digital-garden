Пусть $m$ $\in$ $\mathbb{N}$ и $f$ $\in$ $C^{m-1}(\mathbb{R})$ - $2\pi$-периодичная функция, а $f^{(m)}$ кусочно-непрерывна на $[-\pi; \pi]$, тогда:
1. $c_k[f]$ $=$ $o\left({|k|^{-m}}\right)$, $|k|$ $\to$ $\infty$.
2. $\sup\limits_{x \in \mathbb{R}} |f(x) - S_n[f](x)|$ $=$ $o\left({n^{\frac{1}{2}-m}}\right)$, $n$ $\to$ $\infty$.

#### Доказательство

1. $f^{(m)}$ кусочно-непрерывна на $[-\pi; \pi]$ $\Rightarrow$ она ограничена на всём отрезке за исключением конечного множества точек разрыва меры нуль:$$\exists M > 0 \ : \ a.\forall x \in [-\pi; \pi] \ : \ |f^{(m)}(x)| \le M \Longrightarrow$$$$\Longrightarrow a.\forall x \in [-\pi; \pi] \ : \ |f^{(m)}(x)|^2 \le M^2$$Следовательно: $f^{(m)} \in \mathcal{L}^2_{[-\pi; \pi]}$

2. Вывод связи коэффициентов через интегрирование по частям:$$c_k[f] = \frac{1}{2\pi} \int\limits_{-\pi}^{\pi} f(x) e^{-ikx} dx = \left[ \begin{array}{cc} u = f(x) & dv = e^{-ikx}dx \\ du = f'(x)dx & v = \frac{e^{-ikx}}{-ik} \end{array} \right] = \frac{f(x)e^{-ikx}}{-2\pi ik} \bigg|_{-\pi}^{\pi} + \frac{1}{2\pi ik} \int\limits_{-\pi}^{\pi} f'(x) e^{-ikx} dx = \frac{c_k[f']}{ik}$$
3. Повторное $m$-кратное интегрирование по частям образует цепочку равенств:$$c_k[f] = \frac{c_k[f']}{ik} = \frac{c_k[f'']}{(ik)^2} = \dots = \frac{c_k[f^{(m)}]}{(ik)^m} \quad \Rightarrow \quad |c_k[f]| = \frac{|c_k[f^{(m)}]|}{|k|^m}$$
4. Т.к. $f^{(m)} \in \mathcal{L}^2_{[-\pi; \pi]}$ $\subset$ $\mathcal{L}^1_{[-\pi; \pi]}$, для неё выполнена T. Р-Л об осцилляции:$$\lim\limits_{|k| \to \infty} c_k[f^{(m)}] = 0 \quad \Rightarrow \quad c_k[f^{(m)}] = o(1) \quad \text{при} \quad |k| \to \infty$$$$\Rightarrow |c_k[f]| = \frac{o(1)}{|k|^m} = o\left(|k|^{-m}\right) \quad \text{при} \quad |k| \to \infty$$
5. Пусть $R_n[f](x) = f(x) - S_n[f](x)$ - остаток РФ. По [[Признак Дини|Т. Дини]] $f = \lim\limits_{n \to \infty}S[f]$ и тогда:$$\forall x \in \mathbb{R} \ : \ |f(x) - S_n[f](x)| = \bigg| \sum\limits_{|k| \ge n+1} c_k[f] e^{ikx} \bigg| \le \sum\limits_{|k| \ge n+1} |c_k[f]| = \sum\limits_{|k| \ge n+1} \frac{|c_k[f^{(m)}]|}{|k|^m}$$
6. Применим неравенство КБШ:$$\sum\limits_{|k| \ge n+1} \frac{|c_k[f^{(m)}]|}{|k|^m} \le \sqrt{\sum\limits_{|k| \ge n+1} |c_k[f^{(m)}]|^2} \cdot \sqrt{\sum\limits_{|k| \ge n+1} \frac{1}{k^{2m}}}$$
7. Асимптотический анализ каждого сомножителя при $n \to \infty$ :
    1) По неравенству Бесселя для функции $f^{(m)} \in \mathcal{L}^2_{[-\pi; \pi]}$ сходится числовой ряд из квадратов коэффициентов:$$\sum\limits_{k=-\infty}^{\infty} |c_k[f^{(m)}]|^2 \le \|f^{(m)}\|^2_{2} < \infty \quad \Longrightarrow \quad \sqrt{\sum\limits_{|k| \ge n+1} |c_k[f^{(m)}]|^2} = o(1)$$
    2) Вторая сумма мажорируется интегралом по полупрямой:$$\sum\limits_{k=n+1}^{\infty} \frac{1}{k^{2m}} \le \int\limits_{n}^{\infty} \frac{dt}{t^{2m}} = \frac{1}{(2m-1)n^{2m-1}} = O\left(n^{1-2m}\right) \quad \Longrightarrow \quad \sqrt{\sum\limits_{|k| \ge n+1} \frac{1}{k^{2m}}} = O\left(n^{\frac{1}{2}-m}\right)$$
8. Сшивание оценок даёт искомую скорость равномерной сходимости :$$\sup\limits_{x \in \mathbb{R}} |f(x) - S_n[f](x)| \le o(1) \cdot O\left(n^{\frac{1}{2}-m}\right) = o\left(n^{\frac{1}{2}-m}\right), \quad n \to \infty$$

#### Следствие

Если $m \ge 1$, то $\sup\limits_{x \in \mathbb{R}} |f(x) - S_n[f](x)|$ $=$ $o\left({n^{\frac{1}{2}-m}}\right) \to 0$, $n$ $\to$ $\infty$, т.е. РФ функции $f$ сходится к ней равномерно на $\mathbb{R}$.
