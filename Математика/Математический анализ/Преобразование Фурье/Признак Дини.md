==Опр.== **Условие Дини** (*в точке $x_0$ для функции $f$*) - требование к выполнению следующих условий:
1) $\exists$ конечные односторонние пределы $f(x_0 \pm 0)$
2) $\exists\delta > 0\ :\ \int\limits_0^\delta \frac{|f(x_0 + t) - f(x_0 + 0) + f(x_0 - t) - f(x_0 - 0)|}{t} \, dt < \infty$

==Опр.== **Условие Гёльдера** (*в точке $x_0$ с порядком $\alpha$*) - требование, чтобы $\exists C$ $>$ $0$ $\exists\delta$ $>$ $0$ $\forall x \in U_\delta(x_0)\ :$ $|f(x) - f(x_0)| \le C |x - x_0|^\alpha$

### Утв. Связь условий Дини и Гёльдера

Если $f$ удовлетворяет условию Гёльдера в точке $x_0$ с порядком $\alpha$ $>$ $0$, то для $f$ выполнено условие Дини при.

#### Доказательство
1. Подставим оценку Гёльдера в интеграл условия Дини:$$\int\limits_0^\delta \frac{|f(x_0 + t) + f(x_0 - t) - 2f(x_0)|}{t} \, dt \le \int\limits_0^\delta \frac{|f(x_0 + t) - f(x_0)| + |f(x_0 - t) - f(x_0)|}{t} \, dt$$
2. Применим ==неравенство Гёльдера\*== к числителю:$$\int\limits_0^\delta \frac{C t^\alpha + C t^\alpha}{t} \, dt = 2C \int\limits_0^\delta t^{\alpha - 1} \, dt = \frac{2C}{\alpha} \delta^\alpha$$
3. Т.к. $\alpha$ $>$ $0$, то $\frac{2C}{\alpha} \delta^\alpha < \infty$ $\Rightarrow$ интеграл сходится, т.ч. условие Дини выполнено.

### Теорема Дини

Пусть $f \in \mathcal{L}^1_{[-\pi; \pi]}$. Если для $f$ выполнено условие Дини в точке $x_0 \in [-\pi; \pi]$, то $\lim\limits_{n \to \infty} S_n[f](x_0)$ $=$ $\frac{1}{2}(f(x_0 - 0) + f(x_0 + 0))$

#### Доказательство
1. Из чётности и нормировки ядра Дирихле : $\int\limits_{-\pi}^{\pi} D_n(t) \, dt = 1 \ \Rightarrow \ 2 \int\limits_0^\pi D_n(t) \, dt = 1$. Домножим это равенство на полусумму односторонних пределов:$$\frac{1}{2}(f(x_0 - 0) + f(x_0 + 0)) = (f(x_0 - 0) + f(x_0 + 0)) \int\limits_0^\pi  D_n(t) \, dt = \int\limits_0^\pi (f(x_0 - 0) + f(x_0 + 0)) D_n(t) \, dt$$
2. Формула частичной суммы РФ через ядро Дирихле:$$S_n[f](x) = \int\limits_{-\pi}^\pi f(t) D_n(x-t) dt$$Сделаем замены $u = x - t$ и $v = t - x$:$$S_n[f](x) = -\int\limits_{\pi + x}^{-\pi + x} f(x - u) D_n(u) du = \int\limits_{-\pi}^{\pi} f(x - t) D_n(t) dt$$$$S_n[f](x) = \int\limits_{-\pi-x}^{\pi-x} f(x + v) D_n(-v) dv  = \int\limits_{-\pi}^\pi f(x + t) D_n(t) dt$$
3. Частичную сумму РФ можно получить как полусумму этих двух формул:$$S_n[f](x) = \frac{1}{2}(S_n[f](x) + S_n[f](x)) = \frac{1}{2}\int\limits_{-\pi}^\pi (f(x-t) + f(x + t)) D_n(t) dt$$Что в силу чётности подынтегральной функции равно:$$S_n[f](x) = \int\limits_{0}^\pi (f(x-t) + f(x + t)) D_n(t) dt$$
4. Составим разность под интегралом:$S_n[f](x_0) - \frac{1}{2}(f(x_0 - 0) + f(x_0 + 0)) =$ $\int\limits_0^\pi (f(x_0 + t) + f(x_0 - t) - f(x_0 - 0) + f(x_0 + 0)) D_n(t) \, dt =$ $\int\limits_0^\pi (f(x_0 + t) + f(x_0 - t) - f(x_0 - 0) + f(x_0 + 0)) \frac{\sin\left(\left(n + \frac{1}{2}\right)t\right)}{4\pi \sin\left(\frac{t}{2}\right)} \, dt$. Пусть $g(t) = \frac{1}{t}(f(x_0 + t) + f(x_0 - t) - f(x_0 - 0) + f(x_0 + 0))$, а $\varphi(t) = \frac{t}{4\pi\sin \frac{t}{2}}$. Тогда по условию теоремы $g \in \mathcal{L}^1_{[0; \delta] \cup [\delta; \pi]}$, а $\varphi$ непрерывна на $(0; \pi]$ и $\lim\limits_{t \to 0}\varphi(t) = (2\pi)^{-1}$ $\Rightarrow$ $h(t) = g(t)\varphi(t) \mathbb{1}_{(0; \pi]}(t) \in \mathcal{L}^1(\mathbb{R})$
5. По Т. Р-Л об осцилляции при $n$ $\to$ $\infty$ интеграл $$S_n[f](x_0) - \frac{1}{2}(f(x_0 - 0) + f(x_0 + 0)) = \int\limits_0^\pi h(t) \sin(n + \frac{1}{2})t \cdot dt$$ зануляется $\Rightarrow$ $\lim\limits_{n \to \infty} S_n[f](x_0) = \frac{1}{2}(f(x_0 - 0) + f(x_0 + 0))$.

#### Следствия

1) Если $f \in C(x_0)$, то $\lim\limits_{n \to \infty} S_n[f](x_0)$ $=$ $f(x_0)$.
