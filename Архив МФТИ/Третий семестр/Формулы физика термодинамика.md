## Константы
1) $k_Б = 1.4 \cdot 10^{-23} \frac{Дж}{К}$
2) $R = 8.31 \frac{Дж}{К \cdot моль}$
3) $N_A = 6 \cdot 10^{22}$
4) $P_0 = 10^5 Па = 1\ атм.$
5) $0 \textdegree C = 273 К\ (\Delta 1 \textdegree C = \Delta 1 К)$ 

- $\delta$ - аналог дифференциала для величин, которые не описывают состояние объекта. По ним можно интегрировать. Например, $\Delta Q = \int\limits_{Q_0}^{Q_1} \delta Q$

## Газы
1) Идеальный (примеры: гелий, аргон, азот, воздух, ...) - Количество степеней свободы зависит от формы частицы. 
	1) Одноатомный идеальный газ $i = 3$
	2) Двухатомный идеальный газ или $CO_2$ $i=5$
	3) Остальные идеальные газы $i=6$
2) Ван-дер-Вальса. Для газа должны быть заданы две константы: $a$ и $b$. Для него:
	1) $(P + \frac{a}{V^2})(V - b) = \nu R T$ - *уравнение состояние*
	2) $dU = n c_V dT - \frac{a}{dV}$
	3) $S = nc_V \ln T + n R \ln (V - b) + S_0$

## МКТ
1) $\overline{v} = \sqrt(\overline{v^2}) = \sqrt{\frac{3k_БT}{m}}$ - среднеквадратичная скорость
2) $v_{вер} = \sqrt{\frac{2RT}{\mu}}$  - наиболее вероятная скорость
3) $n = \frac{N}{V} = \frac{N_A \rho}{\mu}$
4) $\nu = \frac{N}{N_A} = \frac{m}{\nu}$
5) $P = nkT$
6) $PV = \nu R T$ - *уравнение состояния идеального газа (уравнение Менделеева-Клайперона)*
7) $dU = \frac{i}{2}\nu R dT$ - *для идеального газа*
8) $\delta A = P dV$
9) $c_v = \frac{i}{2}R$ - *удельная теплоёмкость при постоянном объёме*
10) $c_p = \frac{\delta Q}{dT} = \frac{i+2}{2}R$ - *удельная теплоёмкость при постоянном давлении*
11) $\gamma = \frac{c_p}{c_v}$
12) $c = \sqrt{\frac{\gamma k_Б T}{\mu}}$ - *скорость звука в газе*

## Начала термодинамики (1е и 2е)
1) $\delta Q = dU - A = dU + A_{внеш}$
2) $dS \geq \frac{\delta Q}{T}$ (Для **квазистатического** процесса $dS = \frac{\delta Q}{T}$)

## Энтропия
1) $\widetilde{S} = \ln \widetilde{\Gamma}$ - *общий случай ($\widetilde{\Gamma}$ - количество возможных различных состояний системы)*
2) $S = k_Б \ln \Gamma$ - *для газа (множества независимых частиц) ($\Gamma$ - статический вес)*
3) $S_{universe} = 0$
4) $S = n c_V \ln T  + n R \ln V + S_0$ - *для идеального газа*

## Процессы
1) **Изобарный** - $P = const$
2) **Изохорный** - $V = const$
3) **Изотермический** - $T = const$, уравнение $PV = const$
4) **Адиабатический** - $\delta Q = 0$ *(теплоизолированная система)*, уравнение $PV^{\gamma} = T^{\gamma}P^{-1} = const$
5) **Квазистатический** - медленный, равновесный
6) **Изоэнтропический** - $dS = 0$ $\Leftrightarrow$ и адиабатический, и квазистатический

## Распределения
1) $n(h) = n_0 \cdot e^{-\frac{mgh}{k_БT}}$ - распределение **Больцмана**, применимо для концентрации, плотности и координат в поле силы тяжести
2) $p(v_x) = \frac{1}{\sigma \sqrt{2 \pi}}e^{\frac{-v_x^2}{2\sigma^2}}$, где $\sigma = \sqrt{\frac{k_БT}{m}}$ - **нормальное** распределение, применимо для проекции скорости на любую ось
3) $p(v) = \frac{S_{поверх}}{(\sigma \sqrt{2 \pi})^\frac{i}{2}}e^{\frac{-v^2}{2\sigma^2}}$, где $\sigma = \sqrt{\frac{k_БT}{m}}$, а $S_{поверх}$ - площадь сферы радиуса $v$ в $i$-мерном пространстве. - распределение **Максвелла**, применимо для скорости (по модулю)
	1) На плоскости: $i = 2$, $S_{поверх} = 2\pi v$
	2) В пространстве: $i=3$, $S_{поверх} = 4 \pi v^2$

## Тепловые машины
1) $Q_+ = A + Q_-$ - *ЗСЭ*
2) $\eta = \frac{A}{Q_+} = \frac{Q_+ - Q_-}{Q_+} = \frac{T_+ - T_-}{T_+}$ - *КПД*
3) **Машина Карно** - идеальная тепловая машина, с максимально возможным КПД, для неё верно: $dS = 0$
4) $Q = cm\Delta T$ - нагрев/охлаждение без смены состояния
5) $Q = \lambda m$ - плавление
6) $Q = q m$ - кипение
7) Условие кипения: $p_{пара} = p_{насыщ}$

## Случайные блуждания
1) $\lambda = (4 \pi r_0^2 n)^{-1}$ - длина свободного пробега
2) $\tau = \frac{\lambda}{\overline{v}}$ - период свободного пробега
3) $D = \frac{\lambda^2}{2 \tau}$ - коэффициент диффузии
4) $\kappa = \frac{\lambda^2}{2 \tau} \cdot \frac{i}{2}k_Бn$ - коэффициент теплопроводности
5) $\Delta \overline{r} = \sqrt{2Dt}$ - среднее отклонение за время $t$ (закон $\sqrt{N}$)
6) $j = -D \nabla n$ - поток (в одномерном случае $\nabla n = \frac{dn}{dx}$)
7) $j_E = -\kappa \nabla T$ - поток энергии/тепла (в одномерном случае $\nabla T = \frac{dT}{dx}$)
8) $j_E = \frac{P}{S}$ - мощность на площадь
9) $\Delta x \approx \lambda$ - условие убывание теплопроводности
10) $D = \frac{k_БT}{\beta}$, где при постоянной скорости спускания $u$: $mg = \beta u$ - ТОЛЬКО для крцпных частиц $r_0 > 10^{-8}$ (соотношение Эйнштейна)
11) $\Delta \overline{r} = \sqrt{3Dt}$ - среднее отклонение за время $t$ - ТОЛЬКО для крцпных частиц $r_0 > 10^{-8}$

## Фазовые переходы
1) $\Delta v = \Delta \frac{1}{\rho}$
2) $\frac{dP}{dT} = \frac{q}{T \Delta v}$ - уравнение Клайперона-Клауза
