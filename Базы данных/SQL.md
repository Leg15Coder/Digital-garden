==Опр.== **SQL (Structured Query Langage)** - ***декларативный*** ЯП для управления данными в [[Реляционная модель данных|реляционной]] БД с соответствующей СУБД

### Типы данных

#### Числовые типы
##### Целые числа
	- SMALLINT
	- INTEGER
	- BIGINT

##### Приближённые числа
	- NUMERIC
	- DECIMAL

##### Числа с плавающей точкой
	- REAL
	- FLOAT4
	- DOUBLE PRECISION
	- FLOAT8

#### Логический тип
	 - BOOLEAN

#### Символьные типы
	- VARCHAR(n)
	- CHAR(n)
	- TEXT

#### Тип времени
	- DATE
	- TIME
	- TIMESTAMP
	- TIMEWITHTIMEZONE
	- TIMESTAMPWITHTIMEZONE
	- INTERVAL
